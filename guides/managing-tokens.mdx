# Managing Security Tokens

Once you've created security tokens, you can manage them through the Token Manager interface. This guide covers the essential token management operations.

## Token Operations

### Transferring Tokens

You can transfer tokens to other addresses individually or in bulk:

```typescript
// Single token transfer
transferShare(assetId, shareId, recipientAddress);

// Bulk token transfer
bulkTransfer(assetId, shareIdArray, recipientAddress);
```

In the UI:
1. Navigate to the **Token Manager**
2. Select the token you want to transfer
3. Click **Transfer** and enter recipient address
4. Select individual shares or specify quantity
5. Confirm the transaction

<Warning>
If compliance features like KYC or whitelisting are enabled, transfers will only work to compliant addresses.
</Warning>

### Minting Additional Tokens

For tokens with minting enabled:

```typescript
addToExisting(assetId, amount);
```

In the UI:
1. Select your token
2. Click **Mint**
3. Enter the amount to mint
4. Confirm the transaction

### Burning Tokens

For tokens with burning enabled:

```typescript
bulkBurn(assetId, shareIdArray);
```

In the UI:
1. Select your token
2. Click **Burn**
3. Select the shares to burn
4. Confirm the transaction

## Document Management

Trusset tokens support attaching documents at both the token and individual share level.

### Global Documents

Documents that apply to all shares of a token:

```typescript
updateGlobalDocs(assetId, documentArray);
```

In the UI:
1. Go to the **Documents** tab
2. Click **Add Global Document**
3. Enter document URL or upload file
4. Save changes

### Share-Specific Documents

Documents linked to specific shares:

```typescript
updateLinkedDocs(assetId, shareId, documentArray);
```

In the UI:
1. Go to **Documents** tab
2. Select specific shares
3. Click **Add Linked Document**
4. Enter document URL or upload file
5. Save changes

## Rights and Tags Management

Tags provide a way to add metadata and custom rights to tokens.

### Adding Token Tags

```typescript
updateTags(assetId, tagArray);
```

In the UI:
1. Go to **Rights & Tags** tab
2. Click **Add Tag**
3. Enter tag name and value
4. Save changes

### Share-Specific Tags

```typescript
updateShareTags(assetId, shareId, tagArray);
```

In the UI:
1. Go to **Rights & Tags** tab
2. Select specific shares
3. Click **Add Tag**
4. Enter tag name and value
5. Save changes

<Note>
Tags can represent custom rights like voting powers, dividend rights, or any other metadata you want to track with specific shares.
</Note>

## Transfer Management

### Setting Transfer Limits

Restrict how many transfers can occur within a time period:

```typescript
setMonthlyTransferLimit(assetId, limit);
```

In the UI:
1. Go to **Settings** tab
2. Under Transfer Settings, set Monthly Transfer Limit
3. Save changes

### Setting Maximum Per Wallet

Prevent concentration of tokens:

```typescript
setMaxPerWallet(assetId, maxAmount);
```

In the UI:
1. Go to **Settings** tab
2. Under Wallet Limits, set Maximum Per Wallet
3. Save changes

## Compliance Management

### Updating Whitelist

For tokens with whitelist enabled:

```typescript
updateWhitelist(assetId, addressArray, statusArray);
```

In the UI:
1. Go to **Compliance** tab
2. Click **Manage Whitelist**
3. Add or remove addresses
4. Save changes

### Updating Blacklist

For tokens with blacklist enabled:

```typescript
updateBlacklist(assetId, addressArray, statusArray);
```

In the UI:
1. Go to **Compliance** tab
2. Click **Manage Blacklist**
3. Add or remove addresses
4. Save changes

### Freezing Accounts

For tokens with freeze enabled:

```typescript
freezeAccount(assetId, account, status);
```

In the UI:
1. Go to **Compliance** tab
2. Click **Freeze Account**
3. Enter address to freeze/unfreeze
4. Save changes

## Viewing Token Information

### Basic Information

View token details, supply information, and ownership:

```typescript
getAssetFullDetails(assetId);
```

In the UI:
1. Select your token in the dashboard
2. View the Overview section

### Ownership Information

Get information about who owns which shares:

```typescript
getSharesByOwner(assetId, ownerAddress);
```

In the UI:
1. Go to **Ownership** tab
2. View the ownership distribution chart and table

### Transaction History

View historical transfers and operations:

In the UI:
1. Go to **Activity** tab
2. View the transaction timeline

## Token Management Best Practices

1. **Regular Audits**: Periodically review token ownership and compliance settings

2. **Document Updates**: Keep linked documents current and relevant

3. **Permission Management**: Carefully manage authority addresses

4. **Compliance Monitoring**: Regularly check compliance status of token holders

5. **Backup**: Keep secure backups of any documents or metadata stored off-chain
