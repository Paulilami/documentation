# Getting Started with Trusset

The Trusset Security Token Demo Platform allows you to create, manage, and interact with security tokens on the blockchain. This guide will help you get started with the platform.

## Prerequisites

Before you begin, you'll need:

1. **Web3 Wallet**: The platform is compatible with 450+ wallets. We recommend [MetaMask](https://metamask.io/).

2. **Sepolia Testnet**: Connect your wallet to the Ethereum Sepolia testnet.

3. **Sepolia ETH**: You'll need test ETH for transactions. Get some from:
   - [Google Cloud Faucet](https://cloud.google.com/application/web3/faucet/ethereum/sepolia)
   - [Alchemy Faucet](https://sepoliafaucet.com/)

## Connecting to the Platform

1. Visit [https://demo.trusset.org](https://demo.trusset.org)

2. Click "Connect Wallet" in the top right corner

3. Select your wallet provider or connect using email/social login via our AppKit

<Note>
If you don't have a wallet, you can create a smart wallet instantly using email or social login through our AppKit integration.
</Note>

## Platform Overview

The Trusset Security Token Demo has three main sections:

### 1. Dashboard

The central hub showing your tokens, balances, and recent activity.

### 2. Token Studio

Where you create and configure security tokens with various properties:
- Standard tokens
- Decentralized tokens
- Compliance-focused tokens
- Custom configurations

### 3. Token Manager

Tools for managing existing tokens:
- Transfer tokens
- Add/modify documents
- Configure compliance settings
- Monitor ownership
