A public ERC-1400-based (soon ERC-3643) token generator allowing anyone to create security tokens on-chain and raise capital by selling these tokens. 

<Warning>Actual security tokens can only be issued with an underlying and approved framework, making the token-studio only working for utility-based tokens.</Warning>
