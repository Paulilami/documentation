## Summary

Our `batchVerifyIdentities()` function was stress-tested across 14 batch sizes with 3 iterations each on a Hardhat local network configured to match Ethereum mainnet parameters (30M block gas limit). The test deployed the full UUPS proxy stack identical to production.

## Methodology

42 individual transactions across 14 batch sizes (1 to 500 identities), each executed 3 times for variance analysis. Tests ran on a Hardhat local network with Ethereum mainnet gas parameters (30M block gas limit, identical EVM execution costs). The full UUPS upgradeable proxy was deployed fresh per test suite, matching the production deployment pattern. Each transaction was verified by reading back on-chain storage to confirm all identities were actually written, not just emitted as events. Gas measurements come directly from transaction receipts.

export const BlockUsageChart = () => {
  const data = [
    { size: 1, pct: 0.37 },
    { size: 5, pct: 1.37 },
    { size: 10, pct: 2.63 },
    { size: 25, pct: 6.40 },
    { size: 50, pct: 12.68 },
    { size: 100, pct: 25.24 },

    { size: 400, pct: 100.7 },
    { size: 500, pct: 126.0 },
  ];
  const w = 540;
  const h = 220;
  const pad = { t: 20, r: 20, b: 30, l: 45 };
  const cw = w - pad.l - pad.r;
  const ch = h - pad.t - pad.b;
  const maxY = 140;
  const points = data.map((d, i) => ({
    x: pad.l + (i / (data.length - 1)) * cw,
    y: pad.t + ch - (d.pct / maxY) * ch,

  const line = points.map((p, i) => `${i === 0 ? 'M' : 'L'} ${p.x} ${p.y}`).join(' ');
  const areaPath = `${line} L ${points[points.length-1].x} ${pad.t + ch} L ${points[0].x} ${pad.t + ch} Z`;
  const limitY = pad.t + ch - (100 / maxY) * ch;
  const [hover, setHover] = useState(null);
  return (
    <div className="p-4 border dark:border-white/10 rounded-2xl not-prose">
      <svg viewBox={`0 0 ${w} ${h}`} className="w-full" style={{ maxHeight: 280 }}>
        {[0, 25, 50, 75, 100, 125].map(v => {
          const y = pad.t + ch - (v / maxY) * ch;
          return (
            <g key={v}>
              <line x1={pad.l} y1={y} x2={w - pad.r} y2={y} stroke="currentColor" strokeOpacity={0.08} />
              <text x={pad.l - 6} y={y + 4} textAnchor="end" fill="currentColor" fillOpacity={0.4} fontSize={9} fontFamily="monospace">{v}%</text>
            </g>
          );
        })}
        <line x1={pad.l} y1={limitY} x2={w - pad.r} y2={limitY} stroke="#ef4444" strokeWidth={1.5} strokeDasharray="6 4" strokeOpacity={0.7} />
        <text x={w - pad.r - 2} y={limitY - 5} textAnchor="end" fill="#ef4444" fontSize={9} fontFamily="monospace" fillOpacity={0.8}>100% block limit</text>
        <path d={areaPath} fill="url(#blockAreaGrad)" />
        <path d={line} fill="none" stroke="#4488ff" strokeWidth={2.5} />
        <defs>
          <linearGradient id="blockAreaGrad" x1="0" y1="0" x2="0" y2="1">
            <stop offset="0%" stopColor="#4488ff" stopOpacity="0.15" />
            <stop offset="100%" stopColor="#4488ff" stopOpacity="0" />
          </linearGradient>
        </defs>
        {points.map((p, i) => (
          <g key={i} onMouseEnter={() => setHover(i)} onMouseLeave={() => setHover(null)} style={{ cursor: 'pointer' }}>
            <circle cx={p.x} cy={p.y} r={hover === i ? 6 : 4} fill={p.pct > 100 ? '#ef4444' : p.pct > 80 ? '#f59e0b' : '#4488ff'} />
            {hover === i && (
              <text x={p.x} y={p.y - 12} textAnchor="middle" fill={p.pct > 100 ? '#ef4444' : '#4488ff'} fontSize={11} fontFamily="monospace" fontWeight="bold">
                {p.pct}%
              </text>
            )}
            <text x={p.x} y={pad.t + ch + 16} textAnchor="middle" fill="currentColor" fillOpacity={0.4} fontSize={8} fontFamily="monospace">{p.size}</text>
          </g>
        ))}
      </svg>
      <p className="text-xs text-zinc-500 mt-1">Batch sizes above 350 exceed 88% of the block. At 400+, gas usage crosses the 30M limit and the transaction reverts.</p>
    </div>
  );
};

## Cost per Identity at Various Gas Prices

export const CostPerIdentityChart = () => {
  const sizes = [1, 5, 10, 25, 50, 100, 150, 200, 250, 300, 350];
  const gasPerID = [110190, 82318, 78833, 76746, 76053, 75715, 75609, 75561, 75537, 75524, 75518];
  const ethUsd = 3000;
  const gweiLevels = [
    { gwei: 5, color: '#22c55e', label: '5 gwei' },
    { gwei: 10, color: '#06b6d4', label: '10 gwei' },
    { gwei: 20, color: '#4488ff', label: '20 gwei' },
    { gwei: 50, color: '#f59e0b', label: '50 gwei' },
    { gwei: 100, color: '#ef4444', label: '100 gwei' },
  ];
  const maxCost = (110190 * 100 / 1e9) * ethUsd;
  const w = 540;
  const h = 220;
  const pad = { t: 20, r: 20, b: 30, l: 50 };
  const cw = w - pad.l - pad.r;
  const ch = h - pad.t - pad.b;
  const getX = (i) => pad.l + (i / (sizes.length - 1)) * cw;
  const getY = (cost) => pad.t + ch - (cost / maxCost) * ch;
  const [hover, setHover] = useState(null);
  const ticks = [0, 5, 10, 15, 20, 25, 30, 35];
  return (
    <div className="p-4 border dark:border-white/10 rounded-2xl not-prose">
      <svg viewBox={`0 0 ${w} ${h}`} className="w-full" style={{ maxHeight: 280 }}>
        {ticks.map(v => {
          const y = getY(v);
          if (y < pad.t || y > pad.t + ch) return null;
          return (
            <g key={v}>
              <line x1={pad.l} y1={y} x2={w - pad.r} y2={y} stroke="currentColor" strokeOpacity={0.08} />
              <text x={pad.l - 6} y={y + 4} textAnchor="end" fill="currentColor" fillOpacity={0.4} fontSize={9} fontFamily="monospace">${v}</text>
            </g>
          );
        })}
        {gweiLevels.map(level => {
          const pts = gasPerID.map((g, i) => {
            const cost = (g * level.gwei / 1e9) * ethUsd;
            return { x: getX(i), y: getY(cost), cost };
          });
          const path = pts.map((p, i) => `${i === 0 ? 'M' : 'L'} ${p.x} ${p.y}`).join(' ');
          return (
            <g key={level.gwei}>
              <path d={path} fill="none" stroke={level.color} strokeWidth={2} strokeOpacity={0.8} />
              {pts.map((p, i) => (
                <circle key={i} cx={p.x} cy={p.y} r={2.5} fill={level.color} fillOpacity={0.8} />
              ))}
              <text x={pts[pts.length-1].x + 4} y={pts[pts.length-1].y + 4} fill={level.color} fontSize={8} fontFamily="monospace">{level.label}</text>
            </g>
          );
        })}
        {sizes.map((s, i) => (
          <g key={i}
            onMouseEnter={() => setHover(i)}
            onMouseLeave={() => setHover(null)}
            style={{ cursor: 'pointer' }}
          >
            <line x1={getX(i)} y1={pad.t} x2={getX(i)} y2={pad.t + ch} stroke="currentColor" strokeOpacity={hover === i ? 0.15 : 0} />
            {hover === i && gweiLevels.map(level => {
              const cost = (gasPerID[i] * level.gwei / 1e9) * ethUsd;
              return (
                <circle key={level.gwei} cx={getX(i)} cy={getY(cost)} r={5} fill={level.color} />
              );
            })}
            {hover === i && (
              <text x={getX(i)} y={pad.t - 4} textAnchor="middle" fill="currentColor" fillOpacity={0.7} fontSize={9} fontFamily="monospace">
                batch {s}
              </text>
            )}
            <text x={getX(i)} y={pad.t + ch + 16} textAnchor="middle" fill="currentColor" fillOpacity={0.4} fontSize={8} fontFamily="monospace">{s}</text>
          </g>
        ))}
      </svg>
      <p className="text-xs text-zinc-500 mt-1">USD cost per identity at ETH $3,000 across gas price scenarios. At 20 gwei (typical), a single identity costs ~$0.45.</p>
    </div>
  );
};

## Cost Estimates

export const CostCalculator = () => {
  const [gasPrice, setGasPrice] = useState(20);
  const [ethPrice, setEthPrice] = useState(3000);
  const batches = [
    { size: 50, gas: 3802670 },
    { size: 100, gas: 7571549 },
    { size: 200, gas: 15112373 },
    { size: 250, gas: 18884357 },
    { size: 300, gas: 22657386 },
    { size: 350, gas: 26431352 },
  ];
  const calc = (gas) => {
    const costEth = (gas * gasPrice) / 1e9;
    const costUsd = costEth * ethPrice;
    return { eth: costEth.toFixed(6), usd: costUsd.toFixed(2) };
  };
  return (
    <div className="p-4 border dark:border-white/10 rounded-2xl not-prose space-y-4">
      <div className="grid grid-cols-2 gap-4">
        <div>
          <label className="block text-sm text-zinc-500 mb-1">Gas Price (gwei)</label>
          <input
            type="range" min="3" max="150" value={gasPrice}
            onChange={e => setGasPrice(Number(e.target.value))}
            className="w-full h-2 bg-zinc-800 rounded-lg appearance-none cursor-pointer"
          />
          <span className="text-sm font-mono text-zinc-300">{gasPrice} gwei</span>
        </div>
        <div>
          <label className="block text-sm text-zinc-500 mb-1">ETH Price (USD)</label>
          <input
            type="range" min="1500" max="5000" step="100" value={ethPrice}
            onChange={e => setEthPrice(Number(e.target.value))}
            className="w-full h-2 bg-zinc-800 rounded-lg appearance-none cursor-pointer"
          />
          <span className="text-sm font-mono text-zinc-300">${ethPrice.toLocaleString()}</span>
        </div>
      </div>
      <table className="w-full text-sm">
        <thead>
          <tr className="border-b border-zinc-800">
            <th className="text-left py-2 text-zinc-500">Batch</th>
            <th className="text-right py-2 text-zinc-500">Total Cost</th>
            <th className="text-right py-2 text-zinc-500">Per Identity</th>
          </tr>
        </thead>
        <tbody>
          {batches.map(b => {
            const c = calc(b.gas);
            const perID = calc(Math.round(b.gas / b.size));
            return (
              <tr key={b.size} className={`border-b border-zinc-800/50 ${b.size === 300 ? 'bg-green-500/5' : ''}`}>
                <td className="py-2 font-mono">{b.size} identities{b.size === 300 ? ' *' : ''}</td>
                <td className="text-right py-2 font-mono">${c.usd}</td>
                <td className="text-right py-2 font-mono">${perID.usd}</td>
              </tr>
            );
          })}
        </tbody>
      </table>
      <p className="text-xs text-zinc-500">* Recommended maximum batch size for Ethereum L1.</p>
    </div>
  );
};

## Additional Findings

<AccordionGroup>
  <Accordion title="Gas variance is near-zero">
    Across 3 iterations per batch size, the coefficient of variation was 0.0001-0.0012%. Gas costs are fully deterministic and predictable for any given batch size. There is no randomness in cost.
  </Accordion>
  <Accordion title="Invalid entries are gracefully skipped">
    Batches containing invalid entries (duplicates, malformed data) do not revert. Invalid entries are skipped and returned in the `failedIndices` array, while all valid entries in the same batch are processed normally.
  </Accordion>
  <Accordion title="Calldata cost breakdown">
    At 500 identities, calldata alone costs ~1,044,212 gas (80,324 bytes). This is a fixed L1 cost that cannot be optimized away. On L2s like Base or Arbitrum, calldata is the dominant cost factor instead of execution.
  </Accordion>
</AccordionGroup>
