The Trusset protocol requires to keep standardized token interfaces to work with all services. These interfaes depend on the underlying asset type / token standard that is used. For ERC-3463-based updates, please visit the official T-Rex documentation. 

<Tip>Updating a smart contract will (legally) require an audit for the changes.</Tip>

Auditing your contract updates is a crucial part of compliant operations.
