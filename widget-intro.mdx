## Overview

The Invest Now widget enables companies to accept investments directly on their website. The widget automatically handles wallet detection, network switching, and investment processing through a fullscreen modal that works across all platforms.

## Quick Start

Add this code to your website:

```html
<script src="https://invest.yourbank.com/widget.js"></script>
<button 
  class="invest-now-btn" 
  data-token="YOUR_TOKEN_ADDRESS" 
  data-instance="YOUR_INSTANCE_ID"
  data-company="Your Company Name"
  style="padding: 12px 24px; background: linear-gradient(90deg, #3b82f6, #6366f1); 
         color: white; border: none; border-radius: 8px; font-size: 16px; 
         font-weight: 600; cursor: pointer;">
  Invest Now
</button>
```

Replace:
- `YOUR_TOKEN_ADDRESS` with your token's contract address
- `YOUR_INSTANCE_ID` with your bank instance ID
- `Your Company Name` with your company name

## How It Works

### Modal Display
The widget injects a fullscreen modal directly into the parent page DOM, breaking out of any iframe constraints. This ensures the investment modal always displays properly regardless of embed container size.

### Button Centering
The button automatically centers itself within any container using flexbox positioning. No manual alignment needed.

### Wallet Detection
The widget automatically detects:
- MetaMask
- Coinbase Wallet
- Brave Wallet
- Rabby Wallet
- WalletConnect compatible wallets

## Wallet Instructions

### No Wallet Detected
When no wallet is found, the widget displays clear instructions:
1. **Desktop**: Install MetaMask browser extension
2. **Mobile**: Use wallet app's built-in browser
3. **Safari**: Use MetaMask iOS app or install extension

### Network Switching
The widget automatically:
1. Detects current network
2. Prompts to switch to Sepolia (testnet)
3. Adds network if not configured

## Customization

### Button Styling
Modify the style attribute to match your brand:

```html
style="padding: 12px 24px; 
       background: #your-color; 
       color: white; 
       border: none; 
       border-radius: 8px; 
       font-size: 16px; 
       font-weight: 600; 
       cursor: pointer;"
```

### Multiple Buttons
Add multiple investment buttons on the same page:

```html
<!-- Product A -->
<button class="invest-now-btn" data-token="0xAAA...">
  Invest in Product A
</button>

<!-- Product B -->
<button class="invest-now-btn" data-token="0xBBB...">
  Invest in Product B
</button>
```

## Security

### Wallet Security
- Never share private keys
- Verify transaction details before confirming
- Check network (Sepolia for testing)

### Integration Security
- Widget runs in sandboxed environment
- No access to parent page data
- Secure communication via postMessage API

## Troubleshooting

### Modal Not Opening
- Check console for errors
- Verify script loaded successfully
- Ensure button has correct data attributes

### Wallet Not Connecting
- Check wallet extension is installed
- Verify wallet is unlocked
- Try refreshing the page

### Transaction Failing
- Ensure sufficient ETH for gas
- Check investment within limits
- Verify sale is active

### Safari Issues
- Use Chrome/Firefox/Brave for best experience
- Or use MetaMask mobile app browser
- Install MetaMask extension if available

## API Endpoints

| Endpoint | Description |
|----------|-------------|

| `/api/v1/widget/validate-investment` | Validate investment amount |
