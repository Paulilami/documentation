# Performance Metrics

Actual gas cost measurements from Trusset protocol testing on Ethereum Sepolia testnet. All operations show constant costs regardless of token quantity.

## Token Creation Costs

| Token Quantity | All Features Enabled | Basic Features Only | EUR Cost* |
|----------------|---------------------|-------------------|-----------|
| 1 Token | 0.0053 SepoliaETH | 0.0053 SepoliaETH | €11.13 |
| 100 Tokens | 0.0053 SepoliaETH | 0.0053 SepoliaETH | €11.13 |
| 10,000 Tokens | 0.0053 SepoliaETH | 0.0053 SepoliaETH | €11.13 |
| 1,000,000 Tokens | 0.0053 SepoliaETH | 0.0053 SepoliaETH | €11.13 |
| 10,000,000 Tokens | 0.0053 SepoliaETH | 0.0053 SepoliaETH | €11.13 |

<Info>
Token creation costs remain constant between 0.0050-0.0058 SepoliaETH regardless of quantity or feature complexity.
</Info>

## Core Operations

| Operation | Token Quantity | Cost (SepoliaETH) | EUR Cost* | Range |
|-----------|----------------|-------------------|-----------|--------|
| **Token Transfer** | 1-10,000,000 | 0.0005 | €1.05 | 0.0003-0.0005 |
| **Token Distribution** | 10 tokens → 10 investors | 0.002 | €4.20 | Stable |
| **Token Distribution** | 2,000 tokens → 20 investors | 0.002 | €4.20 | Stable |
| **Tagging Tokens** | Any quantity | 0.0001 | €0.21 | - |
| **Document Upload** | Per document | 0.0003 | €0.63 | - |
| **Account Freezing** | Per account | 0.0008 | €1.68 | - |

**Key Achievement**: Gas costs remain constant regardless of operation scale. Creating 1 token costs the same as creating 10 million tokens.
