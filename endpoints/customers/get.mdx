---
title: 'Get Customer'
api: 'GET /customers/api/manage/{identifier}'
description: 'Retrieve a single customer by ID or wallet address'
---

Returns a customer record by its internal ID or wallet address. The lookup is scoped to the authenticated instance.

## Path Parameters

<ParamField path="identifier" type="string" required>
  Customer ID (e.g., `cust_abc123`) or Ethereum wallet address.
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">Request status</ResponseField>
<ResponseField name="data" type="object">Full customer object</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/customers/api/manage/0xAbC123dEf456789012345678901234567890AbCd" \
  -H "X-API-Key: trusset_abc123xy_secret..."
```
```typescript TypeScript
const response = await fetch(
  'https://api.trusset.org/customers/api/manage/0xAbC123dEf456789012345678901234567890AbCd',
  { headers: { 'X-API-Key': 'trusset_abc123xy_secret...' } }
);
const { data: customer } = await response.json();
```
</RequestExample>
