---
title: 'Create Customer'
api: 'POST /customers/api/manage'
description: 'Register a new customer for the authenticated instance'
---

Creates a new customer record tied to the authenticated instance. The request body is passed to the customer service for validation and storage.

## Request Body

<ParamField body="walletAddress" type="string" required>
  Ethereum wallet address. Must be a valid checksummed or lowercased address.
</ParamField>

<ParamField body="country" type="string" required>
  ISO 3166-1 alpha-2 or alpha-3 country code (e.g., `DE`, `DEU`).
</ParamField>

<ParamField body="investorType" type="string" default="RETAIL">
  Investor classification. Values: `RETAIL`, `PROFESSIONAL`, `INSTITUTIONAL`.
</ParamField>

<ParamField body="email" type="string">
  Customer email address.
</ParamField>

<ParamField body="metadata" type="object">
  Arbitrary key-value metadata to store with the customer record.
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">Request status</ResponseField>
<ResponseField name="data" type="object">The created customer object</ResponseField>

```typescript TypeScript
const response = await fetch('https://api.trusset.org/customers/api/manage', {
  method: 'POST',
```
