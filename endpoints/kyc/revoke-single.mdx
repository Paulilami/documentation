---
title: 'Revoke Single Identity'
api: 'DELETE /v1/kyc/revoke'
---

Revoke a user's KYC verification, removing their verified status on-chain.

<ParamField body="address" type="string" required>
  Ethereum address of the user to revoke
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE "https://trusser-issuer-backend.vercel.app/v1/kyc/revoke" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6"
  }'
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/v1/kyc/revoke', {
  method: 'DELETE',
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    address: '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6'
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.delete(
    'https://trusser-issuer-backend.vercel.app/v1/kyc/revoke',
    headers={
        'x-api-key': 'YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'address': '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6'
    }
)

data = response.json()
```
