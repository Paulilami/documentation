---
title: 'Get Identity Status'
api: 'GET /v1/kyc/status/{address}'
---

Retrieve the current KYC verification status for a specific address.

<ParamField path="address" type="string" required>
  Ethereum address to check
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/v1/kyc/status/0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/v1/kyc/status/0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/v1/kyc/status/0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
