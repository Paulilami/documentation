---
title: 'Batch Revoke Identities'
api: 'POST /v1/kyc/batch/revoke'
---

Revoke multiple user KYC verifications in a single operation. Supports CSV and JSON file uploads or direct address submission.

<ParamField body="file" type="file">
  CSV or JSON file containing addresses to revoke (max 500MB)
</ParamField>

<ParamField body="addresses" type="array">
  Array of Ethereum addresses to revoke (alternative to file upload)
</ParamField>

<RequestExample>
```bash cURL
# File upload
curl -X POST "https://trusser-issuer-backend.vercel.app/v1/kyc/batch/revoke" \
  -H "x-api-key: YOUR_API_KEY" \
  -F "file=@addresses.csv"

# Direct addresses
curl -X POST "https://trusser-issuer-backend.vercel.app/v1/kyc/batch/revoke" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "addresses": [
      "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6",
      "0x5aAeb6053f3E94C9b9A09f33669435E7Ef1BeAed",
      "0x223452347789Bef77e34AB890034567890CdEf12"
    ]
  }'
```

```python Python
import requests

# File upload
with open('addresses.csv', 'rb') as f:
    files = {'file': f}
    response = requests.post(
        'https://trusser-issuer-backend.vercel.app/v1/kyc/batch/revoke',
        headers={'x-api-key': 'YOUR_API_KEY'},
        files=files
    )

# Direct addresses
response = requests.post(
    'https://trusser-issuer-backend.vercel.app/v1/kyc/batch/revoke',
    headers={
        'x-api-key': 'YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'addresses': [
            '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6',
            '0x5aAeb6053f3E94C9b9A09f33669435E7Ef1BeAed',
            '0x223452347789Bef77e34AB890034567890CdEf12'
        ]
    }
)

data = response.json()
```
