---
title: 'Verify Single Identity'
api: 'POST /v1/kyc/verify'
---

Verify a single user's identity and store KYC data on-chain.

<ParamField body="address" type="string" required>
  Ethereum address of the user to verify
</ParamField>

<ParamField body="country" type="string" required>
  ISO 3166-1 alpha-2 country code (e.g., "US", "GB", "FR")
</ParamField>

<ParamField body="expiry" type="number">
  Unix timestamp for verification expiry (0 for no expiry)
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/v1/kyc/verify" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6",
    "country": "US",
    "expiry": 1735689600
  }'
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/v1/kyc/verify', {
  method: 'POST',

```

```python Python
import requests

response = requests.post(
    'https://trusser-issuer-backend.vercel.app/v1/kyc/verify',
    headers={
        'x-api-key': 'YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'address': '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6',
        'country': 'US',
        'expiry': 1735689600
    }
)

data = response.json()
```
