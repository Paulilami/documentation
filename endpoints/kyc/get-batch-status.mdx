---
title: 'Get Batch Status'
api: 'GET /v1/kyc/batch/{batchId}'
---

Retrieve the processing status and details of a KYC batch operation.

<ParamField path="batchId" type="string" required>
  Batch identifier returned from batch verify or revoke operations
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/v1/kyc/batch/batch_7f8a9b0c1d2e3f4a" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/v1/kyc/batch/batch_7f8a9b0c1d2e3f4a', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/v1/kyc/batch/batch_7f8a9b0c1d2e3f4a',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
</RequestExample>
