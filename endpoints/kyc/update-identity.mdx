---
title: 'Update Identity'
api: 'PUT /v1/kyc/update'
---

Update an existing user's KYC information including country and expiry date.

<ParamField body="address" type="string" required>
  Ethereum address of the user to update
</ParamField>

<ParamField body="country" type="string" required>
  New ISO 3166-1 alpha-2 country code
</ParamField>

<ParamField body="expiry" type="number">
  New Unix timestamp for verification expiry (0 for no expiry)
</ParamField>

<RequestExample>
```bash cURL
curl -X PUT "https://trusser-issuer-backend.vercel.app/v1/kyc/update" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6",
    "country": "GB",
    "expiry": 1767225600
  }'
```
