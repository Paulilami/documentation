---
title: 'Batch Verify Identities'
api: 'POST /v1/kyc/batch/verify'
---

Verify multiple user identities in a single operation. Supports CSV and JSON file uploads or direct data submission.

<ParamField body="file" type="file">
  CSV or JSON file containing user addresses and country codes (max 500MB)
</ParamField>

<ParamField body="data" type="array">
  Array of user verification objects (alternative to file upload)
</ParamField>

<ParamField body="expiry" type="number">
  Unix timestamp for verification expiry (applies to all records)
</ParamField>

<RequestExample>
```bash cURL
# File upload
curl -X POST "https://trusser-issuer-backend.vercel.app/v1/kyc/batch/verify" \
  -H "x-api-key: YOUR_API_KEY" \
  -F "file=@kyc_data.csv" \
  -F "expiry=1735689600"

# Direct data
curl -X POST "https://trusser-issuer-backend.vercel.app/v1/kyc/batch/verify" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "data": [
      {"address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6", "country": "US"},
      {"address": "0x5aAeb6053f3E94C9b9A09f33669435E7Ef1BeAed", "country": "GB"}
    ],
    "expiry": 1735689600
  }'
```

```python Python
import requests

# File upload
with open('kyc_data.csv', 'rb') as f:
    files = {'file': f}
    data = {'expiry': '1735689600'}
    response = requests.post(
        'https://trusser-issuer-backend.vercel.app/v1/kyc/batch/verify',
        headers={'x-api-key': 'YOUR_API_KEY'},
        files=files,
        data=data
    )

# Direct data
response = requests.post(
    'https://trusser-issuer-backend.vercel.app/v1/kyc/batch/verify',
    headers={
        'x-api-key': 'YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'data': [
            {'address': '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb6', 'country': 'US'},
            {'address': '0x5aAeb6053f3E94C9b9A09f33669435E7Ef1BeAed', 'country': 'GB'}
        ],
        'expiry': 1735689600
    }
)

data = response.json()
```
