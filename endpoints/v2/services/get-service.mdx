---
title: 'Get Service Details'
api: 'GET /api/v2/services/{serviceId}'
---

Retrieve detailed information about a specific service, including its configuration and available capabilities.

## Path Parameters

<ParamField path="serviceId" type="string" required>
  The service identifier (e.g., `tokenization`, `gold`, `kyc`, `customers`)
</ParamField>

## Response Fields

<ResponseField name="id" type="string">Service identifier</ResponseField>
<ResponseField name="name" type="string">Human-readable service name</ResponseField>
<ResponseField name="enabled" type="boolean">Whether service is active</ResponseField>
<ResponseField name="endpoints" type="array">List of available API endpoints</ResponseField>
<ResponseField name="stats" type="object">Service-specific usage statistics</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.trusset.org/api/v2/services/tokenization" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const serviceId = 'tokenization';

const response = await fetch(
  `https://api.trusset.org/api/v2/services/${serviceId}`,
  {
    method: 'GET',
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
console.log(`Service: ${data.data.name}`);
console.log(`Enabled: ${data.data.enabled}`);
console.log(`Endpoints:`, data.data.endpoints);
```
```python Python
import requests

service_id = 'tokenization'

response = requests.get(
    f'https://api.trusset.org/api/v2/services/{service_id}',
    headers={
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
)

data = response.json()
print(f"Service: {data['data']['name']}")
print(f"Enabled: {data['data']['enabled']}")
print(f"Endpoints: {data['data']['endpoints']}")
```
```typescript TypeScript
interface ServiceDetailResponse {
  success: boolean;
  data: {
    id: string;
    name: string;
    enabled: boolean;
    endpoints: string[];
    stats?: Record<string, number>;
  };
  metadata: {
    requestId: string;
    timestamp: string;
  };
}

const serviceId = 'tokenization';
const response = await fetch(
  `https://api.trusset.org/api/v2/services/${serviceId}`,
  {
    headers: { 'x-api-key': 'trusset_v2_abc123xy_...' }
  }
);
const data: ServiceDetailResponse = await response.json();
```
