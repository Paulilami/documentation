---
title: 'Get Token Holders'
api: 'GET /api/v2/custody/api/tokens/{tokenAddress}/holders'
---

Retrieve all customers holding a balance of a specific token.

## Path Parameters

<ParamField path="tokenAddress" type="string" required>
  Token contract address
</ParamField>

## Query Parameters

<ParamField query="limit" type="number" default="100">
  Maximum results to return (max 500)
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of results to skip for pagination
</ParamField>

## Response Fields

<ResponseField name="token" type="object">Token information</ResponseField>
<ResponseField name="holders" type="array">Array of customer balances</ResponseField>
<ResponseField name="total" type="number">Total holder count</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.trusset.org/api/v2/custody/api/tokens/0xAbC123dEf456789012345678901234567890AbCd/holders?limit=50" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const tokenAddress = '0xAbC123dEf456789012345678901234567890AbCd';

const response = await fetch(
  `https://api.trusset.org/api/v2/custody/api/tokens/${tokenAddress}/holders?limit=50`,
  {
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
console.log(`${data.data.total} holders found`);
```
```python Python
import requests

token_address = '0xAbC123dEf456789012345678901234567890AbCd'

response = requests.get(
    f'https://api.trusset.org/api/v2/custody/api/tokens/{token_address}/holders',
    headers={
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    },
    params={'limit': 50}
)

data = response.json()
print(f"{data['data']['total']} holders found")
```
