---
title: 'Get Token'
api: 'GET /api/v2/custody/api/tokens/{tokenAddress}'
---

Retrieve details for a specific registered token.

## Path Parameters

<ParamField path="tokenAddress" type="string" required>
  Token contract address
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.trusset.org/api/v2/custody/api/tokens/0xAbC123dEf456789012345678901234567890AbCd" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const tokenAddress = '0xAbC123dEf456789012345678901234567890AbCd';

const response = await fetch(
  `https://api.trusset.org/api/v2/custody/api/tokens/${tokenAddress}`,
  {
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
console.log('Token:', data.data.name);
```
```python Python
import requests

token_address = '0xAbC123dEf456789012345678901234567890AbCd'

response = requests.get(
    f'https://api.trusset.org/api/v2/custody/api/tokens/{token_address}',
    headers={
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
)

data = response.json()
print(f"Token: {data['data']['name']}")
```
