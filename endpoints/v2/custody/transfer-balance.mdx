---
title: 'Transfer Balance'
api: 'POST /api/v2/custody/api/balances/transfer'
---

Transfer token balance between two customers. This is an atomic operation that debits the sender and credits the recipient in a single transaction.

## Request Body

<ParamField body="tokenAddress" type="string" required>
  Token contract address
</ParamField>

<ParamField body="fromCustomerRef" type="string" required>
  Sender customer identifier
</ParamField>

<ParamField body="toCustomerRef" type="string" required>
  Recipient customer identifier
</ParamField>

<ParamField body="amount" type="string" required>
  Amount to transfer
</ParamField>

<ParamField body="reference" type="string">
  External reference ID
</ParamField>

<ParamField body="toCustomerMetadata" type="object">
  Metadata for recipient (used if creating new balance record)
</ParamField>

<ParamField body="transactionMetadata" type="object">
  Metadata to store with the transaction
</ParamField>

## Response Fields

<ResponseField name="fromCustomerRef" type="string">Sender identifier</ResponseField>
<ResponseField name="toCustomerRef" type="string">Recipient identifier</ResponseField>
<ResponseField name="tokenAddress" type="string">Token address</ResponseField>
<ResponseField name="amount" type="string">Transferred amount</ResponseField>
<ResponseField name="transactionId" type="string">Created transaction ID</ResponseField>
