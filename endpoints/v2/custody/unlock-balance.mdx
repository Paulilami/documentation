---
title: 'Unlock Balance'
api: 'POST /api/v2/custody/api/balances/unlock'
---

Release previously locked tokens, making them available for transfer or debit operations again.

## Request Body

<ParamField body="tokenAddress" type="string" required>
  Token contract address
</ParamField>

<ParamField body="customerRef" type="string" required>
  Customer identifier
</ParamField>

<ParamField body="amount" type="string" required>
  Amount to unlock
</ParamField>

<ParamField body="reference" type="string">
  Unlock reason or reference ID
</ParamField>

<ParamField body="transactionMetadata" type="object">
  Metadata to store with the transaction
</ParamField>

## Response Fields

<ResponseField name="customerRef" type="string">Customer identifier</ResponseField>
<ResponseField name="tokenAddress" type="string">Token address</ResponseField>
<ResponseField name="lockedBalance" type="string">New locked balance</ResponseField>
<ResponseField name="transactionId" type="string">Created transaction ID</ResponseField>

```javascript JavaScript
const response = await fetch('https://api.trusset.org/api/v2/custody/api/balances/unlock', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
  },
  body: JSON.stringify({
    tokenAddress: '0xAbC123dEf456789012345678901234567890AbCd',
    customerRef: 'CUST-12345',
    amount: '50.0',
    reference: 'SETTLEMENT-COMPLETE-001'
  })
});

const data = await response.json();
console.log(`Remaining locked: ${data.data.lockedBalance}`);
```
