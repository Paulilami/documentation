---
title: 'List Tokens'
api: 'GET /api/v2/custody/api/tokens'
---

Retrieve all tokens registered for custodial tracking in your instance.

## Query Parameters

<ParamField query="activeOnly" type="boolean" default="true">
  If `true`, only returns active tokens
</ParamField>

<ParamField query="limit" type="number" default="100">
  Maximum results to return (max 500)
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of results to skip for pagination
</ParamField>

## Response Fields

<ResponseField name="tokens" type="array">Array of registered tokens</ResponseField>
<ResponseField name="total" type="number">Total count of tokens matching filters</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.trusset.org/api/v2/custody/api/tokens?activeOnly=true&limit=50" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const params = new URLSearchParams({
  activeOnly: 'true',
  limit: '50'
});

const response = await fetch(
  `https://api.trusset.org/api/v2/custody/api/tokens?${params}`,
  {
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
console.log(`Found ${data.data.total} tokens`);
```
```python Python
import requests

response = requests.get(
    'https://api.trusset.org/api/v2/custody/api/tokens',
    headers={
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    },
    params={
        'activeOnly': 'true',
        'limit': 50
    }
)

data = response.json()
print(f"Found {data['data']['total']} tokens")
```
