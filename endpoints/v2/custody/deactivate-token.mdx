---
title: 'Deactivate Token'
api: 'DELETE /api/v2/custody/api/tokens/{tokenAddress}'
---

Deactivate a token from custodial tracking. This does not delete existing balance records but prevents new operations on this token.

<Warning>
  Deactivating a token will cause all balance operations (credit, debit, transfer) to fail for this token until it is re-registered.
</Warning>

## Path Parameters

<ParamField path="tokenAddress" type="string" required>
  Token contract address to deactivate
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.trusset.org/api/v2/custody/api/tokens/0xAbC123dEf456789012345678901234567890AbCd" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const tokenAddress = '0xAbC123dEf456789012345678901234567890AbCd';

const response = await fetch(
  `https://api.trusset.org/api/v2/custody/api/tokens/${tokenAddress}`,
  {
    method: 'DELETE',
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
console.log('Token deactivated:', !data.data.isActive);
```
```python Python
import requests

token_address = '0xAbC123dEf456789012345678901234567890AbCd'

response = requests.delete(
    f'https://api.trusset.org/api/v2/custody/api/tokens/{token_address}',
    headers={
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
)

data = response.json()
print(f"Token deactivated: {not data['data']['isActive']}")
```
