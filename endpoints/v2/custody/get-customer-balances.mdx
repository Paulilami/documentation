---
title: 'Get Customer Balances'
api: 'GET /api/v2/custody/api/balances/{customerRef}'
---

Retrieve all token balances for a specific customer.

## Path Parameters

<ParamField path="customerRef" type="string" required>
  Customer identifier
</ParamField>

## Query Parameters

<ParamField query="limit" type="number" default="100">
  Maximum results to return (max 500)
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of results to skip
</ParamField>

## Response Fields

<ResponseField name="customerRef" type="string">Customer identifier</ResponseField>
<ResponseField name="balances" type="array">Array of token balances with token details</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.trusset.org/api/v2/custody/api/balances/CUST-12345" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const customerRef = 'CUST-12345';

const response = await fetch(
  `https://api.trusset.org/api/v2/custody/api/balances/${customerRef}`,
  {
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
data.data.balances.forEach(b => {
  console.log(`${b.token.symbol}: ${b.availableBalance} available`);
});
```
```python Python
import requests

customer_ref = 'CUST-12345'

response = requests.get(
    f'https://api.trusset.org/api/v2/custody/api/balances/{customer_ref}',
    headers={
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
)

data = response.json()
for b in data['data']['balances']:
    print(f"{b['token']['symbol']}: {b['availableBalance']} available")
```
