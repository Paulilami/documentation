---
title: 'Get Balance'
api: 'GET /api/v2/custody/api/balances/{customerRef}/{tokenAddress}'
---

Retrieve a customer's balance for a specific token.

## Path Parameters

<ParamField path="customerRef" type="string" required>
  Customer identifier
</ParamField>

<ParamField path="tokenAddress" type="string" required>
  Token contract address
</ParamField>

## Response Fields

<ResponseField name="customerRef" type="string">Customer identifier</ResponseField>
<ResponseField name="tokenAddress" type="string">Token address</ResponseField>
<ResponseField name="token" type="object">Token details (name, symbol, decimals, metadataURI)</ResponseField>
<ResponseField name="balance" type="string">Total balance</ResponseField>
<ResponseField name="lockedBalance" type="string">Locked portion</ResponseField>
<ResponseField name="availableBalance" type="string">Available for transfer/debit</ResponseField>
<ResponseField name="metadata" type="object">Customer-specific metadata</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.trusset.org/api/v2/custody/api/balances/CUST-12345/0xAbC123dEf456789012345678901234567890AbCd" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const customerRef = 'CUST-12345';
const tokenAddress = '0xAbC123dEf456789012345678901234567890AbCd';

const response = await fetch(
  `https://api.trusset.org/api/v2/custody/api/balances/${customerRef}/${tokenAddress}`,
  {
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
console.log(`Balance: ${data.data.balance} ${data.data.token.symbol}`);
console.log(`Available: ${data.data.availableBalance}`);
```
```python Python
import requests

customer_ref = 'CUST-12345'
token_address = '0xAbC123dEf456789012345678901234567890AbCd'

response = requests.get(
    f'https://api.trusset.org/api/v2/custody/api/balances/{customer_ref}/{token_address}',
    headers={
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
)

data = response.json()
print(f"Balance: {data['data']['balance']} {data['data']['token']['symbol']}")
```

  { headers: { 'x-api-key': 'trusset_v2_abc123xy_...' } }

const data: BalanceResponse = await response.json();
