---
title: 'Credit Balance'
api: 'POST /api/v2/custody/api/balances/credit'
---

Increase a customer's token balance. Use this when a customer deposits, purchases, or otherwise acquires tokens.

## Request Body

<ParamField body="tokenAddress" type="string" required>
  Token contract address
</ParamField>

<ParamField body="customerRef" type="string" required>
  Customer identifier (max 256 characters)
</ParamField>

<ParamField body="amount" type="string" required>
  Amount to credit (human-readable, e.g., "100.5")
</ParamField>

<ParamField body="reference" type="string">
  External reference ID for reconciliation
</ParamField>

<ParamField body="customerMetadata" type="object">
  Metadata to store with the customer balance record
</ParamField>

<ParamField body="transactionMetadata" type="object">
  Metadata to store with the transaction record
</ParamField>

## Response Fields

<ResponseField name="customerRef" type="string">Customer identifier</ResponseField>
<ResponseField name="tokenAddress" type="string">Token address</ResponseField>
<ResponseField name="balance" type="string">New total balance</ResponseField>
<ResponseField name="transactionId" type="string">Created transaction ID</ResponseField>
