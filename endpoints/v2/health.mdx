---
title: 'Health Check'
api: 'GET /api/v2/health'
---

Check the health status of the v2 API. This endpoint does not require authentication.

<RequestExample>
```bash cURL
curl -X GET "https://api.trusset.org/api/v2/health"
```
```javascript JavaScript
const response = await fetch('https://api.trusset.org/api/v2/health');
const data = await response.json();
console.log(data);
```
```python Python
import requests

response = requests.get('https://api.trusset.org/api/v2/health')
data = response.json()
print(data)
```
```typescript TypeScript
const response = await fetch('https://api.trusset.org/api/v2/health');
const data: HealthResponse = await response.json();
```
</RequestExample>
