---
title: 'Check Is Controller'
api: 'GET /api/v2/light-tokens/is-controller/{tokenAddress}/{account}'
---

Check if an address has controller privileges on a Light Token.

## Path Parameters

<ParamField path="tokenAddress" type="string" required>
  Address of the Light Token contract
</ParamField>

<ParamField path="account" type="string" required>
  Address to check for controller status
</ParamField>

## Response Fields

<ResponseField name="account" type="string">Checked account address</ResponseField>
<ResponseField name="isController" type="boolean">Whether the account is a controller</ResponseField>
