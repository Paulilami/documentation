---
title: 'Get Tokens by IVC'
api: 'GET /api/v2/light-tokens/by-ivc/{ivcAddress}'
---

Retrieve all Light Tokens governed by a specific Instance Verification Contract (IVC).

## Path Parameters

<ParamField path="ivcAddress" type="string" required>
  Address of the Instance Verification Contract
</ParamField>

## Response Fields

<ResponseField name="tokens" type="array">
  List of token addresses governed by this IVC
</ResponseField>
<ResponseField name="count" type="number">Total number of tokens</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.trusset.org/api/v2/light-tokens/by-ivc/0x1234567890123456789012345678901234567890" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const response = await fetch(
  'https://api.trusset.org/api/v2/light-tokens/by-ivc/0x1234567890123456789012345678901234567890',
  {
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }

const data = await response.json();
console.log('Tokens governed by IVC:', data.data.tokens);
```
