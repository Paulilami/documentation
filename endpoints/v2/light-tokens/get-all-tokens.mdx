---
title: 'Get All Tokens'
api: 'GET /api/v2/light-tokens/all'
---

Retrieve all Light Tokens deployed via the factory contract.

<Info>
  This returns all tokens across all IVCs. Use [Get Tokens by IVC](/endpoints/v2/light-tokens/get-tokens-by-ivc) or [Get Instance Tokens](/endpoints/v2/light-tokens/get-instance-tokens) for filtered results.
</Info>

## Response Fields

<ResponseField name="tokens" type="array">
  List of all deployed token addresses
</ResponseField>
<ResponseField name="count" type="number">Total number of tokens</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.trusset.org/api/v2/light-tokens/all" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const response = await fetch(
  'https://api.trusset.org/api/v2/light-tokens/all',
  {
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
console.log('Total tokens:', data.data.count);
```
```python Python
import requests

response = requests.get(
    'https://api.trusset.org/api/v2/light-tokens/all',
    headers={
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
)

data = response.json()
print(f"Total tokens: {data['data']['count']}")
```
