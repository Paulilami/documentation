---
title: 'Controller Transfer'
api: 'POST /api/v2/light-tokens/controller-transfer'
---

Force a transfer of tokens as a controller. This bypasses normal transfer restrictions and is used for regulatory compliance, legal enforcement, or error correction.

<Warning>
  Controller transfers are powerful operations that should only be used when legally required. All controller transfers are logged for audit purposes.
</Warning>

## Request Body

<ParamField body="tokenAddress" type="string" required>
  Address of the Light Token contract
</ParamField>

<ParamField body="from" type="string" required>
  Address to transfer tokens from
</ParamField>

<ParamField body="to" type="string" required>
  Address to transfer tokens to
</ParamField>

<ParamField body="amount" type="string" required>
  Amount of tokens to transfer
</ParamField>

<ParamField body="data" type="string">
  Optional data explaining the reason for the forced transfer (hex-encoded)
</ParamField>

<ParamField body="operatorData" type="string">
  Optional operator data for compliance tracking (hex-encoded)
</ParamField>

<ParamField body="useRelayer" type="boolean" default="false">
  If `true`, IVC executes the transaction. If `false`, returns unsigned transaction data
</ParamField>
