---
title: 'Create Light Token'
api: 'POST /api/v2/light-tokens/create'
---

Deploy a new ERC-1400 compliant Light Token via the factory contract. The token is automatically linked to an IVC (Instance Verification Contract) for governance.

## Request Body

<ParamField body="name" type="string" required>
  Full name of the token (e.g., "Acme Corporation Shares")
</ParamField>

<ParamField body="symbol" type="string" required>
  Token symbol, typically 3-5 characters (e.g., "ACME")
</ParamField>

<ParamField body="decimals" type="number" required>
  Number of decimal places. Use `0` for indivisible tokens (shares), `18` for divisible assets
</ParamField>

<ParamField body="ivcAddress" type="string" required>
  Address of the Instance Verification Contract that will govern this token
</ParamField>

<ParamField body="metadataURI" type="string">
  URI pointing to token metadata (IPFS, HTTPS, or on-chain)
</ParamField>

<ParamField body="useRelayer" type="boolean" default="false">
  If `true`, the IVC executes the transaction. If `false`, returns unsigned transaction data for frontend signing
</ParamField>

## Response Fields

<Tabs>
  <Tab title="Relayer Execution">
    <ResponseField name="executed" type="boolean">Always `true`</ResponseField>
    <ResponseField name="tokenAddress" type="string">Deployed token contract address</ResponseField>
    <ResponseField name="transactionHash" type="string">Transaction hash</ResponseField>
    <ResponseField name="blockNumber" type="number">Block number of deployment</ResponseField>
    <ResponseField name="gasUsed" type="string">Gas consumed</ResponseField>
  </Tab>
  <Tab title="Frontend Signing">
    <ResponseField name="executed" type="boolean">Always `false`</ResponseField>
    <ResponseField name="to" type="string">Factory contract address</ResponseField>
    <ResponseField name="data" type="string">Encoded transaction data</ResponseField>
    <ResponseField name="gasLimit" type="string">Recommended gas limit</ResponseField>
    <ResponseField name="description" type="string">Human-readable description</ResponseField>
  </Tab>
</Tabs>

import requests

    'https://api.trusset.org/api/v2/light-tokens/create',

        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
