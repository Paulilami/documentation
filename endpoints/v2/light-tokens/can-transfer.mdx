---
title: 'Check Can Transfer'
api: 'GET /api/v2/light-tokens/can-transfer'
---

Validate whether a transfer can be executed before attempting it. Returns the transfer validity status and any restriction codes.

## Query Parameters

<ParamField query="tokenAddress" type="string" required>
  Address of the Light Token contract
</ParamField>

<ParamField query="from" type="string" required>
  Sender address
</ParamField>

<ParamField query="to" type="string" required>
  Recipient address
</ParamField>

<ParamField query="amount" type="string" required>
  Amount to transfer
</ParamField>

<ParamField query="partition" type="string">
  Partition to transfer from (optional)
</ParamField>

## Response Fields

<ResponseField name="canTransfer" type="boolean">Whether the transfer can be executed</ResponseField>
<ResponseField name="statusCode" type="string">ERC-1400 status code (bytes1)</ResponseField>
<ResponseField name="reasonCode" type="string">Reason code (bytes32)</ResponseField>
<ResponseField name="reason" type="string">Human-readable reason if transfer is blocked</ResponseField>
