---
title: 'Get Balance'
api: 'GET /api/v2/light-tokens/balance/{tokenAddress}/{account}'
---

Retrieve the token balance for a specific account. Optionally query balance for a specific partition.

## Path Parameters

<ParamField path="tokenAddress" type="string" required>
  Address of the Light Token contract
</ParamField>

<ParamField path="account" type="string" required>
  Account address to check balance for
</ParamField>

## Query Parameters

<ParamField query="partition" type="string">
  Specific partition to query. If not provided, returns total balance across all partitions
</ParamField>

## Response Fields

<ResponseField name="account" type="string">Account address</ResponseField>
<ResponseField name="balance" type="string">Token balance</ResponseField>
<ResponseField name="partition" type="string">Partition queried (if specified)</ResponseField>
<ResponseField name="partitions" type="array">
  List of partitions the account holds tokens in (when no specific partition is queried)
  <Expandable title="partition item">
    <ResponseField name="partition" type="string">Partition identifier (bytes32)</ResponseField>
    <ResponseField name="balance" type="string">Balance in this partition</ResponseField>
  </Expandable>
</ResponseField>
<ResponseField name="frozen" type="boolean">Whether the account is frozen</ResponseField>
