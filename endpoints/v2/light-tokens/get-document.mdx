---
title: 'Get Document'
api: 'GET /api/v2/light-tokens/documents/{tokenAddress}/{documentName}'
---

Retrieve a specific compliance document by name from a Light Token.

## Path Parameters

<ParamField path="tokenAddress" type="string" required>
  Address of the Light Token contract
</ParamField>

<ParamField path="documentName" type="string" required>
  Name/identifier of the document to retrieve
</ParamField>

## Response Fields

<ResponseField name="name" type="string">Document identifier</ResponseField>
<ResponseField name="uri" type="string">Document URI</ResponseField>
<ResponseField name="documentHash" type="string">Content hash for verification</ResponseField>
<ResponseField name="lastModified" type="number">Unix timestamp of last modification</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.trusset.org/api/v2/light-tokens/documents/0xAbC123dEf456789012345678901234567890AbCd/prospectus" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const response = await fetch(
  'https://api.trusset.org/api/v2/light-tokens/documents/0xAbC123dEf456789012345678901234567890AbCd/prospectus',
  {
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
console.log('Document URI:', data.data.uri);
```
```python Python
import requests

response = requests.get(
    'https://api.trusset.org/api/v2/light-tokens/documents/0xAbC123dEf456789012345678901234567890AbCd/prospectus',
    headers={
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
)

data = response.json()
print(f"Document URI: {data['data']['uri']}")
```
