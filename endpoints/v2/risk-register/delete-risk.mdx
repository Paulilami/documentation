---
title: 'Delete Risk Record'
api: 'DELETE /api/v2/risk/assets/{assetAddress}'
---

Permanently delete a risk record and its history.

<Warning>
  This action is permanent. All historical data will be deleted. Consider updating the risk score instead if you need to maintain audit trails.
</Warning>

## Path Parameters

<ParamField path="assetAddress" type="string" required>
  The token contract address
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.trusset.org/api/v2/risk/assets/0xAbC123dEf456789012345678901234567890AbCd" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const response = await fetch(
  'https://api.trusset.org/api/v2/risk/assets/0xAbC123dEf456789012345678901234567890AbCd',
  {
    method: 'DELETE',
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
if (data.success) {
  console.log('Risk record deleted');
}
```
