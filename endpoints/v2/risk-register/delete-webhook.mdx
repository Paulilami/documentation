---
title: 'Delete Webhook Subscription'
api: 'DELETE /api/v2/risk/webhooks/{subscriptionId}'
---

Remove a webhook subscription.

## Path Parameters

<ParamField path="subscriptionId" type="string" required>
  The ID of the webhook subscription to delete
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.trusset.org/api/v2/risk/webhooks/clxwebhook001" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const subscriptionId = 'clxwebhook001';

const response = await fetch(
  `https://api.trusset.org/api/v2/risk/webhooks/${subscriptionId}`,
  {
    method: 'DELETE',
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
if (data.success) {
  console.log('Webhook subscription deleted');
}
```
```python Python
import requests

subscription_id = 'clxwebhook001'

response = requests.delete(
    f'https://api.trusset.org/api/v2/risk/webhooks/{subscription_id}',
    headers={
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
)

data = response.json()
if data['success']:
    print('Webhook subscription deleted')
```
