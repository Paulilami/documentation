---
title: 'List Risk Records'
api: 'GET /api/v2/risk/assets'
---

List all risk records for the instance with optional filtering.

## Query Parameters

<ParamField query="category" type="string">
  Filter by risk category: `credit`, `market`, `liquidity`, `operational`, `regulatory`, `counterparty`
</ParamField>

<ParamField query="minScore" type="number">
  Minimum risk score (inclusive)
</ParamField>

<ParamField query="maxScore" type="number">
  Maximum risk score (inclusive)
</ParamField>

<ParamField query="grade" type="string">
  Filter by letter grade (A-F)
</ParamField>

<ParamField query="limit" type="number" default="100">
  Maximum records to return (max 100)
</ParamField>

<ParamField query="offset" type="number" default="0">
  Pagination offset
</ParamField>

## Response Fields

<ResponseField name="records" type="array">Array of risk records</ResponseField>
<ResponseField name="total" type="number">Total matching records</ResponseField>
<ResponseField name="limit" type="number">Applied limit</ResponseField>
<ResponseField name="offset" type="number">Applied offset</ResponseField>

```javascript JavaScript
const params = new URLSearchParams({
  category: 'credit',
  maxScore: '50',
  limit: '20'
});

const response = await fetch(
  `https://api.trusset.org/api/v2/risk/assets?${params}`,
  {
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
console.log(`Found ${data.data.total} low-risk credit assets`);
```

import requests
