---
title: 'List Webhook Subscriptions'
api: 'GET /api/v2/risk/webhooks'
---

List all webhook subscriptions for the instance.

<Note>
  Webhook secrets are masked in responses. Store your secret when creating the subscription.
</Note>

<RequestExample>
```bash cURL
curl -X GET "https://api.trusset.org/api/v2/risk/webhooks" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const response = await fetch(
  'https://api.trusset.org/api/v2/risk/webhooks',
  {
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
data.data.forEach(sub => {
  console.log(`${sub.id}: ${sub.endpoint} (${sub.active ? 'active' : 'inactive'})`);
});
```
```python Python
import requests

response = requests.get(
    'https://api.trusset.org/api/v2/risk/webhooks',
    headers={
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
)

data = response.json()
for sub in data['data']:
    status = 'active' if sub['active'] else 'inactive'
    print(f"{sub['id']}: {sub['endpoint']} ({status})")
```
```typescript TypeScript
interface WebhookSubscription {
  id: string;
  endpoint: string;
  secret: string;
  assetAddress: string | null;
  events: string[];
  active: boolean;
  lastTriggered: string | null;
  failureCount: number;
}

const response = await fetch(
  'https://api.trusset.org/api/v2/risk/webhooks',
  { headers: { 'x-api-key': 'trusset_v2_abc123xy_...' } }
);
const data = await response.json();
const subscriptions: WebhookSubscription[] = data.data;
```
</RequestExample>
