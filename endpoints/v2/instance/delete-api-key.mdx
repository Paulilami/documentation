---
title: 'Delete API Key'
api: 'DELETE /api/v2/instance/api-keys/{keyId}'
---

Permanently delete an API key from the instance.

<Warning>
  This action is permanent and cannot be undone. Consider revoking instead if you may need to audit past usage.
</Warning>

## Path Parameters

<ParamField path="keyId" type="string" required>
  The ID of the API key to delete
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.trusset.org/api/v2/instance/api-keys/key_abc123def456" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..."
```
```javascript JavaScript
const keyId = 'key_abc123def456';

const response = await fetch(
  `https://api.trusset.org/api/v2/instance/api-keys/${keyId}`,
  {
    method: 'DELETE',
    headers: {
      'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
  }
);

const data = await response.json();
if (data.success) {
  console.log('Key deleted successfully');
}
```
```python Python
import requests

key_id = 'key_abc123def456'

response = requests.delete(
    f'https://api.trusset.org/api/v2/instance/api-keys/{key_id}',
    headers={
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    }
)

data = response.json()
if data['success']:
    print('Key deleted successfully')
```
