---
title: 'Create API Key'
api: 'POST /api/v2/instance/api-keys'
---

Create a new v2 API key for the current instance. The full key is only returned once during creation.

<Warning>
  Store the returned API key securely. It will not be shown again after this response.
</Warning>

## Request Body

<ParamField body="name" type="string" required>
  Display name for the API key (max 100 characters)
</ParamField>

## Response Fields

<ResponseField name="id" type="string">Unique key identifier</ResponseField>
<ResponseField name="key" type="string">The full API key (only shown once)</ResponseField>
<ResponseField name="name" type="string">Display name</ResponseField>
<ResponseField name="version" type="number">Always `2` for new keys</ResponseField>
<ResponseField name="status" type="string">Always `ACTIVE` for new keys</ResponseField>
<ResponseField name="createdAt" type="string">Creation timestamp</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.trusset.org/api/v2/instance/api-keys" \
  -H "Content-Type: application/json" \
  -H "x-api-key: trusset_v2_abc123xy_a1b2c3d4e5f6..." \
  -d '{
    "name": "Backend Service Key"
  }'
```
```javascript JavaScript
const response = await fetch('https://api.trusset.org/api/v2/instance/api-keys', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
  },
  body: JSON.stringify({
    name: 'Backend Service Key'
  })
});

const data = await response.json();
console.log('New API Key:', data.data.key);
```
```python Python
import requests

response = requests.post(
    'https://api.trusset.org/api/v2/instance/api-keys',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'trusset_v2_abc123xy_a1b2c3d4e5f6...'
    },
    json={
        'name': 'Backend Service Key'
    }
)

data = response.json()
print(f"New API Key: {data['data']['key']}")
```
