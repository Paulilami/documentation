---
title: 'Get Dividend Pool Info'
api: 'GET /api/corporate-actions/dividend/pool/{stockISIN}/{paymentToken}'
---

Retrieve information about a dividend distribution pool including total amount, distributed amount, and status.

<ParamField path="stockISIN" type="string" required>
  The ISIN of the stock
</ParamField>

<ParamField path="paymentToken" type="string" required>
  Contract address of the payment token
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/corporate-actions/dividend/pool/US0378331005/0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "x-instance-id: YOUR_INSTANCE_ID"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/corporate-actions/dividend/pool/US0378331005/0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
```
