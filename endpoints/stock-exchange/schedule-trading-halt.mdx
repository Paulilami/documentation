---
title: 'Schedule Trading Halt'
api: 'POST /api/corporate-actions/schedule/trading-halt'
---

Schedule a temporary trading halt for a tokenized stock. This will pause all transfers at the specified time.

<ParamField body="stockISIN" type="string" required>
  The ISIN of the stock to halt
</ParamField>

<ParamField body="scheduledTime" type="string" required>
  ISO 8601 timestamp for when the trading halt should execute
</ParamField>

<ParamField body="externalReference" type="string">
  External reference ID for tracking
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/corporate-actions/schedule/trading-halt" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "x-instance-id: YOUR_INSTANCE_ID" \
  -d '{
    "stockISIN": "US0378331005",
    "scheduledTime": "2025-02-15T14:30:00Z",
    "externalReference": "HALT-NEWS-2025"
  }'
```


