---
title: 'Schedule Delisting'
api: 'POST /api/corporate-actions/schedule/delisting'
---

Schedule a delisting for a tokenized stock. After execution, the token will be permanently disabled.

<ParamField body="stockISIN" type="string" required>
  The ISIN of the stock to delist
</ParamField>

<ParamField body="scheduledTime" type="string" required>
  ISO 8601 timestamp for when the delisting should execute
</ParamField>

<ParamField body="externalReference" type="string">
  External reference ID for tracking
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/corporate-actions/schedule/delisting" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "x-instance-id: YOUR_INSTANCE_ID" \
  -d '{
    "stockISIN": "US0378331005",
    "scheduledTime": "2025-06-01T00:00:00Z",
    "externalReference": "DELIST-2025-001"
  }'
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/corporate-actions/schedule/delisting', {

```

```python Python
import requests

response = requests.post(
    'https://trusser-issuer-backend.vercel.app/api/corporate-actions/schedule/delisting',
    headers={
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_TOKEN',
        'x-instance-id': 'YOUR_INSTANCE_ID'
    },
    json={
        'stockISIN': 'US0378331005',
        'scheduledTime': '2025-06-01T00:00:00Z',
        'externalReference': 'DELIST-2025-001'
    }
)

data = response.json()
```
