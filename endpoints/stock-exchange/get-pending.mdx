---
title: 'Get Pending Actions'
api: 'GET /api/corporate-actions/pending'
---

Retrieve all pending corporate actions for your instance that are scheduled but not yet executed.

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/corporate-actions/pending" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "x-instance-id: YOUR_INSTANCE_ID"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/corporate-actions/pending', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'x-instance-id': 'YOUR_INSTANCE_ID'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/corporate-actions/pending',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN',
        'x-instance-id': 'YOUR_INSTANCE_ID'
    }
)

data = response.json()
```
</RequestExample>
