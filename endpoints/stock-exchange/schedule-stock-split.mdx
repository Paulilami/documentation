---
title: 'Schedule Stock Split'
api: 'POST /api/corporate-actions/schedule/stock-split'
---

Schedule a stock split for a tokenized stock. The split will be executed at the specified time, adjusting token balances according to the ratio.

<ParamField body="stockISIN" type="string" required>
  The ISIN of the stock to split
</ParamField>

<ParamField body="ratio" type="number" required>
  The numerator of the split ratio (e.g., 2 for a 2:1 split)
</ParamField>

<ParamField body="divisor" type="number" required>
  The denominator of the split ratio (e.g., 1 for a 2:1 split)
</ParamField>

<ParamField body="scheduledTime" type="string" required>
  ISO 8601 timestamp for when the split should execute
</ParamField>

<ParamField body="externalReference" type="string">
  External reference ID for tracking
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/corporate-actions/schedule/stock-split" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "x-instance-id: YOUR_INSTANCE_ID" \
  -d '{
    "stockISIN": "US0378331005",
    "ratio": 2,
    "divisor": 1,
    "scheduledTime": "2025-02-01T00:00:00Z",
    "externalReference": "SPLIT-2025-001"
  }'
```
