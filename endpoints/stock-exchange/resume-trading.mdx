---
title: 'Resume Trading'
api: 'POST /api/corporate-actions/trading/{stockISIN}/resume'
---

Resume trading for a tokenized stock that was previously paused. This re-enables all token transfers.

<ParamField path="stockISIN" type="string" required>
  The ISIN of the stock to resume
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/corporate-actions/trading/US0378331005/resume" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "x-instance-id: YOUR_INSTANCE_ID"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/corporate-actions/trading/US0378331005/resume', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'x-instance-id': 'YOUR_INSTANCE_ID'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://trusser-issuer-backend.vercel.app/api/corporate-actions/trading/US0378331005/resume',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN',
        'x-instance-id': 'YOUR_INSTANCE_ID'
    }
)

data = response.json()
```
