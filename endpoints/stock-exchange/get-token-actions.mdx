---
title: 'Get Token Actions'
api: 'GET /api/corporate-actions/token/{stockISIN}'
---

Retrieve all corporate actions for a specific tokenized stock, including scheduled, executed, and cancelled actions.

<ParamField path="stockISIN" type="string" required>
  The ISIN of the stock
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/corporate-actions/token/US0378331005" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "x-instance-id: YOUR_INSTANCE_ID"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/corporate-actions/token/US0378331005', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'x-instance-id': 'YOUR_INSTANCE_ID'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/corporate-actions/token/US0378331005',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN',
        'x-instance-id': 'YOUR_INSTANCE_ID'
    }
)

data = response.json()
```
</RequestExample>
