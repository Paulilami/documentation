---
title: 'Pause Trading'
api: 'POST /api/corporate-actions/trading/{stockISIN}/pause'
---

Pause all trading activity for a tokenized stock. This prevents any transfers of the stock token until trading is resumed.

<ParamField path="stockISIN" type="string" required>
  The ISIN of the stock to pause
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/corporate-actions/trading/US0378331005/pause" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "x-instance-id: YOUR_INSTANCE_ID"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/corporate-actions/trading/US0378331005/pause', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'x-instance-id': 'YOUR_INSTANCE_ID'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://trusser-issuer-backend.vercel.app/api/corporate-actions/trading/US0378331005/pause',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN',
        'x-instance-id': 'YOUR_INSTANCE_ID'
    }
)

data = response.json()
```
