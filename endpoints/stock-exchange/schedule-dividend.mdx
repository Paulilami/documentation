---
title: 'Schedule Dividend'
api: 'POST /api/corporate-actions/schedule/dividend'
---

Schedule a cash dividend distribution for tokenized stock holders. The dividend will be paid in the specified payment token at the scheduled time.

<ParamField body="stockISIN" type="string" required>
  The ISIN of the stock paying dividends
</ParamField>

<ParamField body="amount" type="string" required>
  Total dividend amount to distribute (in payment token units)
</ParamField>

<ParamField body="paymentDate" type="string" required>
  ISO 8601 timestamp for the payment date
</ParamField>

<ParamField body="paymentToken" type="string" required>
  Contract address of the payment token (e.g., USDC)
</ParamField>

<ParamField body="scheduledTime" type="string" required>
  ISO 8601 timestamp for when to execute the dividend
</ParamField>

<ParamField body="externalReference" type="string">
  External reference ID for tracking
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/corporate-actions/schedule/dividend" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "x-instance-id: YOUR_INSTANCE_ID" \
  -d '{
    "stockISIN": "US0378331005",
    "amount": "1000000",
    "paymentDate": "2025-03-15T00:00:00Z",

    "scheduledTime": "2025-03-01T00:00:00Z",
    "externalReference": "DIV-Q1-2025"
  }'
```
