---
title: 'Fund Dividend Pool'
api: 'POST /api/corporate-actions/dividend/pool/fund'
---

Add additional funds to an existing dividend distribution pool. The pool must be active to accept new funds.

<ParamField body="stockISIN" type="string" required>
  The ISIN of the stock
</ParamField>

<ParamField body="paymentToken" type="string" required>
  Contract address of the payment token
</ParamField>

<ParamField body="amount" type="string" required>
  Amount to add to the pool
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/corporate-actions/dividend/pool/fund" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "x-instance-id: YOUR_INSTANCE_ID" \
  -d '{
    "stockISIN": "US0378331005",
    "paymentToken": "0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48",
    "amount": "500000"
  }'
```
