---
title: 'Get Lock Status'
api: 'GET /api/stock-exchange/status/{lockId}'
---

Retrieve the current status and details of a locked stock position.

<ParamField path="lockId" type="string" required>
  The unique identifier of the locked position
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/stock-exchange/status/LOCK-1234567890-abcdef1234567890" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/stock-exchange/status/LOCK-1234567890-abcdef1234567890', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/stock-exchange/status/LOCK-1234567890-abcdef1234567890',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
</RequestExample>
