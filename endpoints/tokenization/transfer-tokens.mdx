---
title: 'Transfer Tokens'
api: 'POST /api/v1/tokenization/transfer'
---

Transfer tokens from the keeper wallet to specified recipients.

<ParamField body="tokenAddress" type="string" required>
  ERC20 token contract address
</ParamField>

<ParamField body="transfers" type="array" required>
  List of transfer instructions
  
  <Expandable title="transfers">
    <ParamField body="to" type="string" required>
      Recipient Ethereum address
    </ParamField>
    
    <ParamField body="amount" type="string" required>
      Amount of tokens to transfer (will be adjusted for decimals)
    </ParamField>
  </Expandable>
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/v1/tokenization/transfer" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "tokenAddress": "0x1234567890123456789012345678901234567890",
    "transfers": [
      {
        "to": "0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4",
        "amount": "1000"
      },
      {
        "to": "0xd2135CfB216b74109775236E36d4b433F1DF507B",
        "amount": "2500"
      }
    ]
  }'
```
