---
title: 'Mint Tokens'
api: 'POST /api/v1/tokenization/mint'
---

Mint new tokens and distribute them to specified recipients.

<ParamField body="tokenAddress" type="string" required>
  ERC20 token contract address
</ParamField>

<ParamField body="recipients" type="array" required>
  List of recipients and amounts
  
  <Expandable title="recipients">
    <ParamField body="address" type="string" required>
      Recipient Ethereum address
    </ParamField>
    
    <ParamField body="amount" type="string" required>
      Amount of tokens to mint (will be adjusted for decimals)
    </ParamField>
  </Expandable>
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/v1/tokenization/mint" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "tokenAddress": "0x1234567890123456789012345678901234567890",
    "recipients": [
      {
        "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4",
        "amount": "50000"
      },
      {
        "address": "0xd2135CfB216b74109775236E36d4b433F1DF507B",
        "amount": "25000"
      }
    ]
  }'
```
