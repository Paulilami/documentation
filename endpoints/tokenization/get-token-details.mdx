---
title: 'Get Token Details'
api: 'GET /api/v1/tokenization/token/:address'
---

Retrieve detailed information about a specific token including configuration and supply metrics.

<ParamField path="address" type="string" required>
  ERC20 token contract address
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/v1/tokenization/token/0x1234567890123456789012345678901234567890" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/v1/tokenization/token/0x1234567890123456789012345678901234567890', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/v1/tokenization/token/0x1234567890123456789012345678901234567890',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
</RequestExample>
