---
title: 'Get Balances'
api: 'GET /api/v1/tokenization/balances/:tokenAddress'
---

Retrieve token balances and compliance status for specified holders.

<ParamField path="tokenAddress" type="string" required>
  ERC20 token contract address
</ParamField>

<ParamField query="holders" type="string">
  Comma-separated list of holder addresses to query
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/v1/tokenization/balances/0x1234567890123456789012345678901234567890?holders=0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4,0xd2135CfB216b74109775236E36d4b433F1DF507B" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const holders = [
  '0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4',
  '0xd2135CfB216b74109775236E36d4b433F1DF507B'
];

const response = await fetch(
  `https://trusser-issuer-backend.vercel.app/api/v1/tokenization/balances/0x1234567890123456789012345678901234567890?holders=${holders.join(',')}`,
  {
    method: 'GET',
    headers: {
      'x-api-key': 'YOUR_API_KEY'
    }
  }
);

const data = await response.json();
```

```python Python
import requests

holders = [
    '0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4',
    '0xd2135CfB216b74109775236E36d4b433F1DF507B'
]

response = requests.get(
    f'https://trusser-issuer-backend.vercel.app/api/v1/tokenization/balances/0x1234567890123456789012345678901234567890',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    },
    params={
        'holders': ','.join(holders)
    }
)

data = response.json()
```
