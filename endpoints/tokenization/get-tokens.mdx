---
title: 'Get Tokens'
api: 'GET /api/v1/tokenization/tokens'
---

Retrieve all tokens created by the authenticated issuer with on-chain data.

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/v1/tokenization/tokens" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/v1/tokenization/tokens', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/v1/tokenization/tokens',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
</RequestExample>
