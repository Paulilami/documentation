---
title: 'List Webhook Partners'
api: 'GET /stocks/api/webhooks/partners'
description: 'Retrieve all registered webhook partners for the instance'
---

Returns all webhook partner registrations associated with the authenticated instance.

## Response Fields

<ResponseField name="success" type="boolean">Request status.</ResponseField>
<ResponseField name="data" type="object[]">
  Array of webhook partner records.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/stocks/api/webhooks/partners" \
  -H "X-API-Key: trusset_abc123xy_secret..."
```
```typescript TypeScript
const response = await fetch(
  'https://api.trusset.org/stocks/api/webhooks/partners',
  {
    headers: { 'X-API-Key': 'trusset_abc123xy_secret...' }
  }
);
const { data } = await response.json();
```
</RequestExample>
