---
title: 'Get Balance'
api: 'GET /stocks/api/fetch/balance/{tokenAddress}/{account}'
description: 'Retrieve the total, transferable, and frozen balance for an account'
---

Returns the token balance breakdown for a specific account. The response separates the total balance into its transferable and frozen components.

## Path Parameters

<ParamField path="tokenAddress" type="string" required>
  Address of the stock token contract.
</ParamField>

<ParamField path="account" type="string" required>
  Ethereum address of the account to query.
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">Request status.</ResponseField>
<ResponseField name="data" type="object">
  <Expandable>
    <ResponseField name="account" type="string">Queried account address.</ResponseField>
    <ResponseField name="total" type="string">Total balance (transferable + frozen).</ResponseField>
    <ResponseField name="transferable" type="string">Balance available for transfers.</ResponseField>
    <ResponseField name="frozen" type="string">Balance locked by freeze operations.</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/stocks/api/fetch/balance/0x1234.../0xAbC123..." \
  -H "X-API-Key: trusset_abc123xy_secret..."
```
```typescript TypeScript
const response = await fetch(
  `https://api.trusset.org/stocks/api/fetch/balance/${tokenAddress}/${account}`,
  {
    headers: { 'X-API-Key': 'trusset_abc123xy_secret...' }
  }
);
const { data } = await response.json();
console.log(`Transferable: ${data.transferable}, Frozen: ${data.frozen}`);
```
</RequestExample>
