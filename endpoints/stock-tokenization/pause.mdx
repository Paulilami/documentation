---
title: 'Pause Token'
api: 'POST /stocks/api/manage/{tokenAddress}/pause'
description: 'Halt all transfers on a stock token contract'
---

Pauses the token contract, preventing all transfers until unpaused. Issuance and redemption are also blocked while paused. Typically used during corporate events, regulatory holds, or emergency situations.

## Path Parameters

<ParamField path="tokenAddress" type="string" required>
  Address of the stock token contract.
</ParamField>

## Request Body

<ParamField body="useRelayer" type="boolean" default="false">
  When `true`, the transaction is signed and submitted by the Trusset relayer.
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">Request status.</ResponseField>
<ResponseField name="data" type="object">Pause result including the transaction hash.</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.trusset.org/stocks/api/manage/0x1234.../pause" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: trusset_abc123xy_secret..." \
  -d '{ "useRelayer": true }'
```
```typescript TypeScript
const response = await fetch(
  `https://api.trusset.org/stocks/api/manage/${tokenAddress}/pause`,
  {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'X-API-Key': 'trusset_abc123xy_secret...'
    },
    body: JSON.stringify({ useRelayer: true })
  }
);
const { data } = await response.json();
```
</RequestExample>
