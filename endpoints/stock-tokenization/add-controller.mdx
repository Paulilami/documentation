---
title: 'Add Controller'
api: 'POST /stocks/api/manage/{tokenAddress}/controller/add'
description: 'Grant the controller role to an address on a stock token'
---

Grants the `CONTROLLER` role to the specified account on-chain. Controllers can execute force transfers, freeze tokens, and pause the contract.

## Path Parameters

<ParamField path="tokenAddress" type="string" required>
  Address of the stock token contract.
</ParamField>

## Request Body

<ParamField body="account" type="string" required>
  Ethereum address to grant the controller role.
</ParamField>

<ParamField body="useRelayer" type="boolean" default="false">
  When `true`, the transaction is signed and submitted by the Trusset relayer.
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">Request status.</ResponseField>
<ResponseField name="data" type="object">Transaction result.</ResponseField>

<RequestExample>
```bash cURL

  -H "Content-Type: application/json" \
  -H "X-API-Key: trusset_abc123xy_secret..." \
  -d '{
    "account": "0x9876543210fedcba9876543210fedcba98765432",
```
