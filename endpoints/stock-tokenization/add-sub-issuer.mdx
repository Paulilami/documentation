---
title: 'Add Sub-Issuer'
api: 'POST /stocks/api/manage/{tokenAddress}/sub-issuer/add'
description: 'Grant the sub-issuer role to an address on a stock token'
---

Grants the `SUB_ISSUER` role to the specified account on-chain. Sub-issuers can mint tokens on behalf of the primary issuer. The token must have been deployed with `allowSubIssuers` enabled.

## Path Parameters

<ParamField path="tokenAddress" type="string" required>
  Address of the stock token contract.
</ParamField>

## Request Body

<ParamField body="account" type="string" required>
  Ethereum address to grant the sub-issuer role.
</ParamField>

<ParamField body="useRelayer" type="boolean" default="false">
  When `true`, the transaction is signed and submitted by the Trusset relayer.
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">Request status.</ResponseField>
<ResponseField name="data" type="object">Transaction result.</ResponseField>

<RequestExample>
```bash cURL

  -H "Content-Type: application/json" \
  -H "X-API-Key: trusset_abc123xy_secret..." \
  -d '{
    "account": "0x9876543210fedcba9876543210fedcba98765432",
```
