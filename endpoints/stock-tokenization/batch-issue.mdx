---
title: 'Batch Issue Tokens'
api: 'POST /stocks/api/tokenization/{tokenAddress}/batch-issue'
description: 'Mint tokens to multiple recipients in a single transaction'
---

Issues tokens to up to 200 recipients in one call. The `recipients` and `amounts` arrays must have matching lengths. Each recipient address is validated individually.

## Path Parameters

<ParamField path="tokenAddress" type="string" required>
  Address of the stock token contract.
</ParamField>

## Request Body

<ParamField body="recipients" type="string[]" required>
  Array of Ethereum addresses to receive tokens. Maximum 200 entries.
</ParamField>

<ParamField body="amounts" type="string[]" required>
  Array of token amounts corresponding to each recipient. Each value must be a positive number string.
</ParamField>

<ParamField body="reason" type="string">
  Optional reason or reference for the batch issuance.
</ParamField>

<ParamField body="useRelayer" type="boolean" default="false">
  When `true`, the transaction is signed and submitted by the Trusset relayer.
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">Request status.</ResponseField>
<ResponseField name="data" type="object">
  Batch issuance result including the transaction hash and number of recipients processed.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.trusset.org/stocks/api/tokenization/0x1234.../batch-issue" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: trusset_abc123xy_secret..." \
  -d '{
    "recipients": [
      "0xAbC123dEf456789012345678901234567890AbCd",
      "0x9876543210fedcba9876543210fedcba98765432"
    ],
    "amounts": ["500", "1500"],
    "reason": "Series A distribution",
    "useRelayer": true
  }'
```
```typescript TypeScript
const response = await fetch(
  `https://api.trusset.org/stocks/api/tokenization/${tokenAddress}/batch-issue`,
  {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'X-API-Key': 'trusset_abc123xy_secret...'
    },
    body: JSON.stringify({
      recipients: [
        '0xAbC123dEf456789012345678901234567890AbCd',
        '0x9876543210fedcba9876543210fedcba98765432'
      ],
      amounts: ['500', '1500'],
      reason: 'Series A distribution',
      useRelayer: true
    })
  }
);
const { data } = await response.json();
```
</RequestExample>
