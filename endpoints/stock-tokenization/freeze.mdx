---
title: 'Freeze Tokens'
api: 'POST /stocks/api/manage/{tokenAddress}/freeze'
description: 'Lock a portion of a holder''s tokens, preventing transfer'
---

Freezes a specified amount of tokens held by an account. Frozen tokens remain in the holder's balance but cannot be transferred until unfrozen. Requires the `CONTROLLER` role on the token contract.

## Path Parameters

<ParamField path="tokenAddress" type="string" required>
  Address of the stock token contract.
</ParamField>

## Request Body

<ParamField body="account" type="string" required>
  Ethereum address of the holder whose tokens will be frozen.
</ParamField>

<ParamField body="amount" type="string" required>
  Number of tokens to freeze. Must be greater than zero.
</ParamField>

<ParamField body="useRelayer" type="boolean" default="false">
  When `true`, the transaction is signed and submitted by the Trusset relayer.
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">Request status.</ResponseField>
<ResponseField name="data" type="object">Freeze result including the transaction hash.</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.trusset.org/stocks/api/manage/0x1234.../freeze" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: trusset_abc123xy_secret..." \
  -d '{
    "account": "0xAbC123dEf456789012345678901234567890AbCd",
    "amount": "500",
    "useRelayer": true
  }'
```
```typescript TypeScript
const response = await fetch(

);
const { data } = await response.json();
```
