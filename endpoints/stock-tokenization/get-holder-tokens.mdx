---
title: 'Get Holder Tokens'
api: 'GET /stocks/api/fetch/holder/{walletAddress}'
description: 'List all stock tokens held by a specific wallet address'
---

Returns all stock tokens that a given wallet holds within the authenticated instance. Useful for building portfolio views.

## Path Parameters

<ParamField path="walletAddress" type="string" required>
  Ethereum wallet address of the holder.
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">Request status.</ResponseField>
<ResponseField name="data" type="object[]">
  Array of token holdings for the wallet.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/stocks/api/fetch/holder/0xAbC123..." \
  -H "X-API-Key: trusset_abc123xy_secret..."
```
```typescript TypeScript
const response = await fetch(
  `https://api.trusset.org/stocks/api/fetch/holder/${walletAddress}`,
  {
    headers: { 'X-API-Key': 'trusset_abc123xy_secret...' }
  }
);
const { data } = await response.json();
```
</RequestExample>
