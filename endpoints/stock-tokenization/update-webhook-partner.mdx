---
title: 'Update Webhook Partner'
api: 'PATCH /stocks/api/webhooks/partners/{partnerId}'
description: 'Update a webhook partner''s configuration'
---

Updates the configuration of a registered webhook partner. Only the owning instance can modify a partner. The webhook URL, if updated, must use HTTPS.

## Path Parameters

<ParamField path="partnerId" type="string" required>
  ID of the webhook partner to update.
</ParamField>

## Request Body

Accepts any combination of updatable fields. Only provided fields are changed.

<ParamField body="name" type="string">
  Updated display name.
</ParamField>

<ParamField body="webhookUrl" type="string">
  Updated HTTPS delivery URL.
</ParamField>

<ParamField body="active" type="boolean">
  Set to `false` to disable webhook delivery without deleting the registration.
</ParamField>

## Response Fields

<ResponseField name="success" type="boolean">Request status.</ResponseField>
<ResponseField name="data" type="object">
  <Expandable>
    <ResponseField name="id" type="string">Partner ID.</ResponseField>
    <ResponseField name="name" type="string">Display name.</ResponseField>
    <ResponseField name="webhookUrl" type="string">Delivery URL.</ResponseField>
    <ResponseField name="active" type="boolean">Whether deliveries are enabled.</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X PATCH "https://api.trusset.org/stocks/api/webhooks/partners/wh_abc123" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: trusset_abc123xy_secret..." \
  -d '{
    "webhookUrl": "https://api.example.com/webhooks/trusset-v2",
    "active": true
  }'
```
```typescript TypeScript
const response = await fetch(
  `https://api.trusset.org/stocks/api/webhooks/partners/${partnerId}`,
  {
    method: 'PATCH',
    headers: {
      'Content-Type': 'application/json',
      'X-API-Key': 'trusset_abc123xy_secret...'
    },
    body: JSON.stringify({
      webhookUrl: 'https://api.example.com/webhooks/trusset-v2',
      active: true
    })
  }
);
const { data } = await response.json();
```
</RequestExample>
