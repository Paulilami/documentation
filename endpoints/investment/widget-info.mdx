---
title: 'Widget Token Info'
api: 'GET /api/v1/widget/token-info/:tokenAddress'
---

Retrieve token sale information for the investment widget.

<Note>
  This endpoint is CORS-enabled and publicly accessible for widget integration.
</Note>

<ParamField path="tokenAddress" type="string" required>
  ERC20 token contract address
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/v1/widget/token-info/0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb0"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/v1/widget/token-info/0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb0', {
  method: 'GET'
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/v1/widget/token-info/0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb0'
)

data = response.json()
```
