---
title: 'Create Market Order'
api: 'POST /api/v1/investment/marketplace/order'
---

Create a sell order on the secondary market.

## Request Body

<ParamField body="tokenAddress" type="string" required>
  ERC20 token contract address
</ParamField>

<ParamField body="sellerAddress" type="string" required>
  Ethereum address of the token seller
</ParamField>

<ParamField body="amount" type="number" required>
  Number of tokens to sell
</ParamField>

<ParamField body="pricePerToken" type="number" required>
  Price per token in EUR
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/v1/investment/marketplace/order" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "x-instance-id: YOUR_INSTANCE_ID" \
  -d '{
    "tokenAddress": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb0",
    "sellerAddress": "0x1234567890123456789012345678901234567890",
    "amount": 1000,
    "pricePerToken": 1.5
  }'
```

```python Python
import requests

response = requests.post(

        'x-api-key': 'YOUR_API_KEY',
        'x-instance-id': 'YOUR_INSTANCE_ID'

    json={
        'tokenAddress': '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb0',
```
