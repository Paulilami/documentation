---
title: 'Freeze Accounts'
api: 'POST /api/v1/investment/freeze'
---

Freeze or unfreeze investor accounts for compliance purposes.

## Request Body

<ParamField body="tokenAddress" type="string" required>
  ERC20 token contract address
</ParamField>

<ParamField body="accounts" type="array" required>
  Array of Ethereum addresses to freeze/unfreeze
</ParamField>

<ParamField body="freeze" type="boolean" default="true">
  Whether to freeze (true) or unfreeze (false) the accounts
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/v1/investment/freeze" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "x-instance-id: YOUR_INSTANCE_ID" \
  -d '{
    "tokenAddress": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb0",
    "accounts": ["0x1234567890123456789012345678901234567890"],
    "freeze": true
  }'
```
