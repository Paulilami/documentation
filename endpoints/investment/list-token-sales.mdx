---
title: 'List Token Sales'
api: 'GET /api/v1/investment/token-sales'
---

Retrieve all token sales for your bank instance.

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/v1/investment/token-sales" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "x-instance-id: YOUR_INSTANCE_ID"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/v1/investment/token-sales', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'x-instance-id': 'YOUR_INSTANCE_ID'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/v1/investment/token-sales',
    headers={
        'x-api-key': 'YOUR_API_KEY',
        'x-instance-id': 'YOUR_INSTANCE_ID'
    }
)

data = response.json()
```
