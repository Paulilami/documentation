---
title: 'Get Token Sale'
api: 'GET /api/v1/investment/token/:tokenAddress'
---

Retrieve detailed information about a specific token sale.

<ParamField path="tokenAddress" type="string" required>
  ERC20 token contract address
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/v1/investment/token/0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb0" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "x-instance-id: YOUR_INSTANCE_ID"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/v1/investment/token/0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb0', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'x-instance-id': 'YOUR_INSTANCE_ID'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/v1/investment/token/0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb0',
    headers={
        'x-api-key': 'YOUR_API_KEY',
        'x-instance-id': 'YOUR_INSTANCE_ID'
    }
)

data = response.json()
```
</RequestExample>
