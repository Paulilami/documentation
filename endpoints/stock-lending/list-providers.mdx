---
title: 'List Liquidity Providers'
api: 'GET /lending-stocks/api/positions/{marketId}/providers'
description: 'List all active liquidity providers for a market'
---

Returns all LPs who currently have an active deposit in the specified market.

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/lending-stocks/api/positions/{marketId}/providers" \
  -H "X-API-Key: trusset_your_key_here"
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true,
  "data": {
    "providers": [
      {
        "id": "lp_001",
        "provider": "0x123...456",
        "amountDeposited": "50000.00",
        "sharesOwned": "50000.00",
        "active": true,
        "depositedAt": "2025-06-01T09:00:00.000Z"
      }
    ]
  }
}
```
</ResponseExample>
