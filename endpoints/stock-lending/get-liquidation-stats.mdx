---
title: 'Get Liquidation Stats'
api: 'GET /lending-stocks/api/liquidations/{marketId}/liquidation-stats'
description: 'Aggregate liquidation statistics from on-chain and database'
---

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/lending-stocks/api/liquidations/{marketId}/liquidation-stats" \
  -H "X-API-Key: trusset_your_key_here"
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true,
  "data": {
    "onChain": {
      "totalLiquidations": 3,
      "pendingLiquidationDebt": "0.000000",
      "totalAuctions": 1
    },
    "database": {
      "pending": 0,
      "settled": 3,
      "stuck": 0
    }
  }
}
```
</ResponseExample>
