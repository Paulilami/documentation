---
title: 'Get Liquidatable Loans'
api: 'GET /lending-stocks/api/liquidations/{marketId}/liquidatable-loans'
description: 'List all loans with health factor below 1.0'
---

Scans all active loans in the market and returns those eligible for liquidation. Health factor values below `1.0` indicate the loan is undercollateralized.

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/lending-stocks/api/liquidations/{marketId}/liquidatable-loans" \
  -H "X-API-Key: trusset_your_key_here"
```
</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,

```
</ResponseExample>
