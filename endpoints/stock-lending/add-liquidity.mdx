---
title: 'Add Liquidity'
api: 'POST /lending-stocks/api/positions/{marketId}/add-liquidity'
description: 'Deposit USDC into the lending pool and receive LP shares'
---

Deposits USDC into the market's liquidity pool. The depositor receives proportional LP shares that accrue interest as borrowers repay. Requires the depositor to be KYC-verified in the IdentityRegistry.

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

## Body Parameters

<ParamField body="amount" type="string" required>USDC amount to deposit (e.g. `"10000"`).</ParamField>
<ParamField body="useRelayer" type="boolean" default="false">Execute via relayer or return calldata.</ParamField>

<Info>
  The first deposit into a new pool requires a minimum of 1,000 USDC.
</Info>

<RequestExample>
```bash cURL
curl -X POST "https://api.trusset.org/lending-stocks/api/positions/{marketId}/add-liquidity" \
  -H "X-API-Key: trusset_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"amount": "10000", "useRelayer": true}'
```
</RequestExample>

<ResponseExample>
```json Relayer Response
{
  "success": true,
  "data": {
    "txHash": "0xabc...def",
    "amount": "10000"
  }
}
```
```json Calldata Response
{

  "data": {
    "action": "SIGN_TRANSACTION",
```

</ResponseExample>
