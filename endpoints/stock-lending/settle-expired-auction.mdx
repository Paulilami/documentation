---
title: 'Settle Expired Auction'
api: 'POST /lending-stocks/api/liquidations/{marketId}/settle-expired-auction'
description: 'Settle a Dutch auction that has passed its end time'
---

Routes remaining collateral from an expired auction to the liquidation router for off-chain sale. Requires `LIQUIDATOR_ROLE`.

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

## Body Parameters

<ParamField body="auctionId" type="integer" required>Auction ID.</ParamField>
<ParamField body="useRelayer" type="boolean" default="false">Execute via relayer or return calldata.</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.trusset.org/lending-stocks/api/liquidations/{marketId}/settle-expired-auction" \
  -H "X-API-Key: trusset_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"auctionId": 1, "useRelayer": true}'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true,
  "data": { "txHash": "0xabc...def", "auctionId": 1 }
}
```
</ResponseExample>
