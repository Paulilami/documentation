---
title: 'Get Loan (On-Chain)'
api: 'GET /lending-stocks/api/positions/{marketId}/loan/{loanId}'
description: 'Read a loan directly from the smart contract'
---

Returns live loan data from the on-chain contract by numeric loan ID.

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>
<ParamField path="loanId" type="integer" required>On-chain loan ID (positive integer).</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/lending-stocks/api/positions/{marketId}/loan/5" \
  -H "X-API-Key: trusset_your_key_here"
```
</RequestExample>
