---
title: 'Get User Loans'
api: 'GET /lending-stocks/api/positions/{marketId}/user-loans/{userAddress}'
description: 'List all active loans for a specific borrower'
---

Returns all active on-chain loans for the given wallet address in the specified market.

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>
<ParamField path="userAddress" type="string" required>Borrower wallet address (0x-prefixed, 40 hex chars).</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/lending-stocks/api/positions/{marketId}/user-loans/0xabc...def" \
  -H "X-API-Key: trusset_your_key_here"
```
</RequestExample>
