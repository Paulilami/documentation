---
title: 'List Active Auctions'
api: 'GET /lending-stocks/api/liquidations/{marketId}/auctions'
description: 'List all active Dutch auctions for a market'
---

Returns auctions that are currently accepting bids. Only applies to markets with `useDutchAuction` enabled.

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/lending-stocks/api/liquidations/{marketId}/auctions" \
  -H "X-API-Key: trusset_your_key_here"
```
</RequestExample>
