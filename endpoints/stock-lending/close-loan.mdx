---
title: 'Close Loan'
api: 'POST /lending-stocks/api/positions/{marketId}/close-loan'
description: 'Close a fully repaid loan and return collateral'
---

Closes a loan that has zero remaining debt and returns all collateral to the borrower. Reverts if any debt remains.

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

## Body Parameters

<ParamField body="loanId" type="integer" required>On-chain loan ID.</ParamField>
<ParamField body="useRelayer" type="boolean" default="false">Execute via relayer or return calldata.</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.trusset.org/lending-stocks/api/positions/{marketId}/close-loan" \
  -H "X-API-Key: trusset_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"loanId": 5, "useRelayer": true}'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true,
  "data": { "txHash": "0xabc...def", "loanId": 5 }
}
```
</ResponseExample>
