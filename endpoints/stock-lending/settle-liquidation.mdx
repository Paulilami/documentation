---
title: 'Settle Liquidation'
api: 'POST /lending-stocks/api/liquidations/settle/{liquidationId}'
description: 'Settle a pending liquidation with USDC proceeds'
---

Routes USDC proceeds from a completed collateral sale back to the lending market through the liquidation router.

<Warning>
  This is a write operation. The relayer must hold sufficient USDC and have `OPERATOR_ROLE` on the router.
</Warning>

## Path Parameters

<ParamField path="liquidationId" type="string" required>Liquidation ID from the router.</ParamField>

## Body Parameters

<ParamField body="usdcProceeds" type="string" required>
  USDC amount to settle (e.g. `"4272.50"`). Must be positive.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.trusset.org/lending-stocks/api/liquidations/settle/4" \
  -H "X-API-Key: trusset_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"usdcProceeds": "4272.50"}'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true,
  "data": {
    "liquidationId": "4",
    "usdcProceeds": "4272.50",
    "txHash": "0xabc...def"
  }
}
```
</ResponseExample>
