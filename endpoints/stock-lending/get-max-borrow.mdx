---
title: 'Get Max Borrow'
api: 'GET /lending-stocks/api/markets/{marketId}/max-borrow'
description: 'Calculate maximum borrowable USDC for a given collateral amount'
---

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

## Query Parameters

<ParamField query="collateralAmount" type="string" required>Collateral token amount (e.g. `"100"`).</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/lending-stocks/api/markets/{marketId}/max-borrow?collateralAmount=100" \
  -H "X-API-Key: trusset_your_key_here"
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true,
  "data": {
    "collateralAmount": "100",
    "maxBorrow": "13912.500000"
  }
}
```
</ResponseExample>
