---
title: 'Get Pending Liquidations'
api: 'GET /lending-stocks/api/liquidations/{marketId}/pending-liquidations'
description: 'List unsettled liquidations for a market'
---

Returns liquidations that have not yet been settled, enriched with on-chain data from the router.

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/lending-stocks/api/liquidations/{marketId}/pending-liquidations" \
  -H "X-API-Key: trusset_your_key_here"
```
</RequestExample>
