---
title: 'List Positions'
api: 'GET /lending-stocks/api/positions/{marketId}/positions'
description: 'List all borrowing positions for a lending market'
---

Returns all positions (loans) recorded for the specified market, ordered newest first.

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/lending-stocks/api/positions/{marketId}/positions" \
  -H "X-API-Key: trusset_your_key_here"
```
</RequestExample>
