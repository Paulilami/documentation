---
title: 'Remove Liquidity'
api: 'POST /lending-stocks/api/positions/{marketId}/remove-liquidity'
description: 'Burn LP shares and withdraw proportional USDC'
---

Redeems LP shares for USDC at the current share price. Reverts if insufficient liquidity is available (e.g. if most USDC is currently lent out).

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

## Body Parameters

<ParamField body="shares" type="string" required>Number of LP shares to redeem (18-decimal formatted, e.g. `"10000"`).</ParamField>
<ParamField body="useRelayer" type="boolean" default="false">Execute via relayer or return calldata.</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.trusset.org/lending-stocks/api/positions/{marketId}/remove-liquidity" \
  -H "X-API-Key: trusset_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"shares": "10000", "useRelayer": true}'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true,
  "data": {
    "txHash": "0xabc...def",
    "sharesRedeemed": "10000"
  }
}
```
</ResponseExample>
