---
title: 'Get Oracle Status'
api: 'GET /lending-stocks/api/markets/{marketId}/oracle-status'
description: 'Retrieve the current price oracle status for a market'
---

Returns the oracle's current price, staleness, age, deviation limits, and whether the relayer signer is authorized.

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/lending-stocks/api/markets/{marketId}/oracle-status" \
  -H "X-API-Key: trusset_your_key_here"
```
</RequestExample>
