---
title: 'Claim Escrowed Collateral'
api: 'POST /lending-stocks/api/positions/{marketId}/claim-escrowed-collateral'
description: 'Claim collateral held in escrow after KYC renewal'
---

When a borrower's KYC expires, collateral returns are held in escrow. After renewing KYC verification, the borrower calls this endpoint to claim their escrowed collateral.

## Path Parameters

<ParamField path="marketId" type="string" required>Market ID.</ParamField>

## Body Parameters

<ParamField body="useRelayer" type="boolean" default="false">Execute via relayer or return calldata.</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.trusset.org/lending-stocks/api/positions/{marketId}/claim-escrowed-collateral" \
  -H "X-API-Key: trusset_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"useRelayer": true}'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true,
  "data": { "txHash": "0xabc...def" }
}
```
</ResponseExample>
