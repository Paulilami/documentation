---
title: 'Sync Pool Data'
api: 'POST /api/lending/pools/{poolAddress}/sync'
---

Manually synchronize on-chain pool data with the database. This updates total assets, borrows, liquidity, and rates.

<ParamField path="poolAddress" type="string" required>
  Address of the lending pool to sync
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/lending/pools/0x1234567890123456789012345678901234567890/sync" \
  -H "x-api-key: YOUR_API_KEY"
  const response = await fetch('https://trusser-issuer-backend.vercel.app/api/lending/pools/0x1234567890123456789012345678901234567890/sync', {
  method: 'POST',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();

import requests

response = requests.post(
    'https://trusser-issuer-backend.vercel.app/api/lending/pools/0x1234567890123456789012345678901234567890/sync',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
</RequestExample>
<ResponseExample>
```json Response
{
  "success": true,
  "message": "Pool data synced successfully"
}
```
</ResponseExample>
