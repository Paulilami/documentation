---
title: 'Get User Analytics'
api: 'GET /api/lending/analytics/user/{userAddress}'
---

Retrieve comprehensive analytics for a specific user including all deposits, borrows, and liquidations.

<ParamField path="userAddress" type="string" required>
  Address of the user to get analytics for
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/lending/analytics/user/0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4" \
  -H "x-api-key: YOUR_API_KEY"
  const response = await fetch('https://trusser-issuer-backend.vercel.app/api/lending/analytics/user/0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();

import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/lending/analytics/user/0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
</RequestExample>
