---
title: 'Create Lending Pool'
api: 'POST /api/lending/pools/create'
---

Create a new lending pool for a specific asset. Pools are usually created automatically when needed, but this endpoint allows manual creation with custom parameters.

<ParamField body="assetAddress" type="string" required>
  Address of the ERC20 token to create a lending pool for
</ParamField>

<ParamField body="collateralAddress" type="string" optional>
  Address of the collateral token (defaults to USDC)
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/lending/pools/create" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "assetAddress": "0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4",
    "collateralAddress": "0x1c7D4B196Cb0C7B01d743Fbc6116a902379C7238"
  }'
javascriptconst response = await fetch('https://trusser-issuer-backend.vercel.app/api/lending/pools/create', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    assetAddress: '0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4',
    collateralAddress: '0x1c7D4B196Cb0C7B01d743Fbc6116a902379C7238'
  })
});

const data = await response.json();
pythonimport requests

response = requests.post(
    'https://trusser-issuer-backend.vercel.app/api/lending/pools/create',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    json={
        'assetAddress': '0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4',
        'collateralAddress': '0x1c7D4B196Cb0C7B01d743Fbc6116a902379C7238'
    }
)

data = response.json()
```
