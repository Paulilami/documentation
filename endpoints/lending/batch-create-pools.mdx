---
title: 'Batch Create Pools'
api: 'POST /api/lending/pools/batch-create'
---

Create multiple lending pools in a single transaction for efficiency. Each pool will be configured with default settings.

<ParamField body="assets" type="array" required>
  Array of ERC20 token addresses to create lending pools for
</ParamField>

<ParamField body="names" type="array" required>
  Array of names for the lending pools (must match assets length)
</ParamField>

<ParamField body="symbols" type="array" required>
  Array of symbols for the lending pool tokens (must match assets length)
</ParamField>
