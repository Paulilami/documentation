---
title: 'Get All Pools'
api: 'GET /api/lending/pools'
---

Retrieve all lending pools or filter by specific asset addresses.

<ParamField query="assets" type="string" optional>
  Comma-separated list of asset addresses to filter pools by
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/lending/pools?assets=0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4,0xd2135CfB216b74109775236E36d4b433F1DF507B" \
  -H "x-api-key: YOUR_API_KEY"
  const response = await fetch('https://trusser-issuer-backend.vercel.app/api/lending/pools?assets=0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4,0xd2135CfB216b74109775236E36d4b433F1DF507B', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/lending/pools?assets=0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4,0xd2135CfB216b74109775236E36d4b433F1DF507B',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
</RequestExample>
