---
title: 'Get Pool Analytics'
api: 'GET /api/lending/analytics/pools'
---

Retrieve analytics data for all lending pools including deposit, borrow, and liquidation counts.

<ParamField query="limit" type="number" optional default="50">
  Maximum number of pools to return
</ParamField>

<ParamField query="offset" type="number" optional default="0">
  Number of pools to skip for pagination
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/lending/analytics/pools?limit=10&offset=0" \
  -H "x-api-key: YOUR_API_KEY"
  const response = await fetch('https://trusser-issuer-backend.vercel.app/api/lending/analytics/pools?limit=10&offset=0', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/lending/analytics/pools?limit=10&offset=0',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
</RequestExample>
