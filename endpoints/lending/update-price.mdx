---
title: 'Update Asset Price'
api: 'POST /api/lending/oracle/update-price'
---

Update the price of an asset in the lending oracle for accurate collateral valuations and health factor calculations.

<ParamField body="assetAddress" type="string" required>
  Address of the asset token to update price for
</ParamField>

<ParamField body="price" type="string" required>
  New price in USD (18 decimal precision)
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/lending/oracle/update-price" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "assetAddress": "0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4",
    "price": "150.50"
  }'
javascriptconst response = await fetch('https://trusser-issuer-backend.vercel.app/api/lending/oracle/update-price', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    assetAddress: '0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4',
    price: '150.50'
  })
});

const data = await response.json();
pythonimport requests

response = requests.post(
    'https://trusser-issuer-backend.vercel.app/api/lending/oracle/update-price',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    json={
        'assetAddress': '0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4',
        'price': '150.50'
    }
)

data = response.json()
```
