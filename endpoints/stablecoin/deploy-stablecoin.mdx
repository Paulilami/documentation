---
title: 'Deploy Stablecoin'
api: 'POST /api/stablecoin/deploy'
---

Deploy a new regulated stablecoin with customizable parameters for compliance and reserve management.

<ParamField body="name" type="string" required>
  Name of the stablecoin (e.g., "Euro Stablecoin")
</ParamField>

<ParamField body="symbol" type="string" required>
  Token symbol (e.g., "EURS")
</ParamField>

<ParamField body="decimals" type="number" default="6">
  Number of decimal places (typically 6 for stablecoins)
</ParamField>

<ParamField body="kycRequired" type="boolean" default="false">
  Enable mandatory KYC verification for all transfers
</ParamField>

<ParamField body="reserveDataURI" type="string" default="">
  URI pointing to reserve attestation documents
</ParamField>

<ParamField body="transferFeeBps" type="number" default="0">
  Transfer fee in basis points (1 bps = 0.01%, max 1000 bps = 10%)
</ParamField>

<ParamField body="feeRecipient" type="string">
  Address to receive transfer fees (defaults to admin if not specified)
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/stablecoin/deploy" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "name": "Euro Stablecoin",
    "symbol": "EURS",
    "decimals": 6,
    "kycRequired": true,
    "reserveDataURI": "https://reserves.example.com/eurs",
    "transferFeeBps": 50,
    "feeRecipient": "0x9876543210987654321098765432109876543210"
  }'
```
