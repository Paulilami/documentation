---
title: 'Mint Tokens'
api: 'POST /api/stablecoin/mint'
---

Mint new stablecoin tokens to a specified address. Requires minter role.

<ParamField body="stablecoinAddress" type="string" required>
  Address of the stablecoin contract
</ParamField>

<ParamField body="to" type="string" required>
  Recipient wallet address
</ParamField>

<ParamField body="amount" type="string" required>
  Amount of tokens to mint
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/stablecoin/mint" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "stablecoinAddress": "0xf265EB7723d5493722e92C9dBF4E993d7b36D903",
    "to": "0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4",
    "amount": "10000"
  }'
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/stablecoin/mint', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    stablecoinAddress: '0xf265EB7723d5493722e92C9dBF4E993d7b36D903',
    to: '0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4',
    amount: '10000'
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://trusser-issuer-backend.vercel.app/api/stablecoin/mint',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    json={
        'stablecoinAddress': '0xf265EB7723d5493722e92C9dBF4E993d7b36D903',
        'to': '0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4',
        'amount': '10000'
    }
)

data = response.json()
```
