---
title: 'Check Account Roles'
api: 'GET /api/stablecoin/roles/{stablecoinAddress}/{account}'
---

Check which roles an account has for a specific stablecoin.

<ParamField path="stablecoinAddress" type="string" required>
  Address of the stablecoin contract
</ParamField>

<ParamField path="account" type="string" required>
  Address to check roles for
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/stablecoin/roles/0xf265EB7723d5493722e92C9dBF4E993d7b36D903/0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4" \
  -H "x-api-key: YOUR_API_KEY"
```
