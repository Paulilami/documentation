---
title: 'Get Stablecoin Information'
api: 'GET /api/stablecoin/info/{stablecoinAddress}'
---

Retrieve detailed information about a stablecoin including supply metrics, reserve data, and fee configuration.

<ParamField path="stablecoinAddress" type="string" required>
  Address of the stablecoin contract
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/stablecoin/info/0xBDdEB75beb9A604dE5C82495f1331E0A4E7b2f15" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/stablecoin/info/0xBDdEB75beb9A604dE5C82495f1331E0A4E7b2f15', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/stablecoin/info/0xBDdEB75beb9A604dE5C82495f1331E0A4E7b2f15',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
