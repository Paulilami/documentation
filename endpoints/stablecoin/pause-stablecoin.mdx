---
title: 'Pause Stablecoin'
api: 'POST /api/stablecoin/pause'
---

Pause all stablecoin operations for emergency situations. Requires admin role.

<ParamField body="stablecoinAddress" type="string" required>
  Address of the stablecoin contract
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/stablecoin/pause" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "stablecoinAddress": "0xf265EB7723d5493722e92C9dBF4E993d7b36D903"
  }'
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/stablecoin/pause', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    stablecoinAddress: '0xf265EB7723d5493722e92C9dBF4E993d7b36D903'
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://trusser-issuer-backend.vercel.app/api/stablecoin/pause',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    json={
        'stablecoinAddress': '0xf265EB7723d5493722e92C9dBF4E993d7b36D903'
    }
)

data = response.json()
```
