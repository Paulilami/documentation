---
title: 'Get Transaction History'
api: 'GET /api/stablecoin/transactions/{stablecoinAddress}'
---

Retrieve transaction history for a stablecoin including mints and burns.

<ParamField path="stablecoinAddress" type="string" required>
  Address of the stablecoin contract
</ParamField>

<ParamField query="type" type="string">
  Filter by transaction type: "MINT" or "BURN"
</ParamField>

<ParamField query="limit" type="number" default="50">
  Maximum number of results to return
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of results to skip
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/stablecoin/transactions/0xf265EB7723d5493722e92C9dBF4E993d7b36D903?type=MINT&limit=10" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/stablecoin/transactions/0xf265EB7723d5493722e92C9dBF4E993d7b36D903?type=MINT&limit=10', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/stablecoin/transactions/0xf265EB7723d5493722e92C9dBF4E993d7b36D903',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    },
    params={
        'type': 'MINT',
        'limit': 10
    }
)

data = response.json()
```
