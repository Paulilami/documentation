---
title: 'List Stablecoins'
api: 'GET /api/stablecoin/list'
---

List all stablecoins deployed by the issuer with pagination support.

<ParamField query="limit" type="number" default="50">
  Maximum number of results to return
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of results to skip
</ParamField>

<RequestExample>

```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/stablecoin/list?limit=10&offset=0" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/stablecoin/list?limit=10&offset=0', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/stablecoin/list',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    },
    params={
        'limit': 10,
        'offset': 0
    }
)

data = response.json()
```
