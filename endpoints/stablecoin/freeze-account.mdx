---
title: 'Freeze Account'
api: 'POST /api/stablecoin/freeze'
---

Freeze an account to prevent all transfers for regulatory compliance. Requires freezer role.

<ParamField body="stablecoinAddress" type="string" required>
  Address of the stablecoin contract
</ParamField>

<ParamField body="account" type="string" required>
  Address to freeze
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/stablecoin/freeze" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "stablecoinAddress": "0xBDdEB75beb9A604dE5C82495f1331E0A4E7b2f15",
    "account": "0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4"
  }'
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/stablecoin/freeze', {
```
