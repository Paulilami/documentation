---
title: 'Set Mint Limit'
api: 'POST /api/stablecoin/set-mint-limit'
---

Set monthly minting limit for an address to ensure regulatory compliance. Requires admin role.

<ParamField body="stablecoinAddress" type="string" required>
  Address of the stablecoin contract
</ParamField>

<ParamField body="minter" type="string" required>
  Address to set mint limit for
</ParamField>

<ParamField body="limit" type="string" required>
  Monthly mint limit amount (set to "0" for unlimited)
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/stablecoin/set-mint-limit" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "stablecoinAddress": "0xBDdEB75beb9A604dE5C82495f1331E0A4E7b2f15",
    "minter": "0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4",
    "limit": "1000000"
  }'
```
