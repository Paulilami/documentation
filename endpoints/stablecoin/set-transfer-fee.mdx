---
title: 'Set Transfer Fee'
api: 'POST /api/stablecoin/set-transfer-fee'
---

Configure transfer fees for stablecoin transactions. Maximum fee is 10% (1000 basis points). Requires admin role.

<ParamField body="stablecoinAddress" type="string" required>
  Address of the stablecoin contract
</ParamField>

<ParamField body="transferFeeBps" type="number" required>
  Transfer fee in basis points (1 bps = 0.01%, max 1000 bps = 10%)
</ParamField>

<RequestExample>

```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/stablecoin/set-transfer-fee" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "stablecoinAddress": "0xBDdEB75beb9A604dE5C82495f1331E0A4E7b2f15",
    "transferFeeBps": 50
  }'
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/stablecoin/set-transfer-fee', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    stablecoinAddress: '0xBDdEB75beb9A604dE5C82495f1331E0A4E7b2f15',
    transferFeeBps: 50
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://trusser-issuer-backend.vercel.app/api/stablecoin/set-transfer-fee',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY'
    },
    json={
        'stablecoinAddress': '0xBDdEB75beb9A604dE5C82495f1331E0A4E7b2f15',
        'transferFeeBps': 50
    }
)

data = response.json()
```
