---
title: 'Get Leverage Positions'
api: 'GET /api/balancer/v1/leverage/positions'
---

Get all leverage positions for the authenticated user.

<ParamField query="status" type="string" default="open">
  Filter positions by status (open, closed, liquidated)
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/balancer/v1/leverage/positions?status=open" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/balancer/v1/leverage/positions?status=open', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/balancer/v1/leverage/positions',
    params={'status': 'open'},
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)
```
</RequestExample>
