---
title: 'Get Swap Quote'
api: 'POST /api/balancer/v1/swap/quote'
---

Get a price quote for a potential swap without executing the transaction.

<ParamField body="poolId" type="string" required>
  The unique identifier of the pool
</ParamField>

<ParamField body="tokenIn" type="string" required>
  Address of the token being sold
</ParamField>

<ParamField body="tokenOut" type="string" required>
  Address of the token being bought
</ParamField>

<ParamField body="amountIn" type="string" required>
  Amount of tokens to sell
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/balancer/v1/swap/quote" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "poolId": "0x8b1e5c3d4a2f6b9e7c5d3a2f1b8e9c7d5a3f2b1e8c9d7a5f3b2e1c8d9a7f5b3e2c",
    "tokenIn": "0x1234567890123456789012345678901234567890",
    "tokenOut": "0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48",
    "amountIn": "10"
  }'
```
