---
title: 'Get Pool Price'
api: 'GET /api/oracle/market-data/{symbol}'
---

Get current market price data for a stock from the unified oracle system.

<ParamField path="symbol" type="string" required>
  Stock ticker symbol (e.g., "AAPL", "MSFT")
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/oracle/market-data/AAPL" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/oracle/market-data/AAPL', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/oracle/market-data/AAPL',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)
```
</RequestExample>
