---
title: 'Get Pool Price'
api: 'GET /api/balancer-exchange/pools/{poolId}/price'
---

Get the current price and liquidity information for a specific pool.

<ParamField path="poolId" type="string" required>
  The unique identifier of the pool
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/balancer-exchange/pools/0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef12/price" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/balancer-exchange/pools/0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef12/price', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/balancer-exchange/pools/0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef12/price',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }

data = response.json()
```
