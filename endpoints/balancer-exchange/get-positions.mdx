---
title: 'Get Positions'
api: 'GET /api/balancer/leverage/positions/{address}'
---

Get all open leveraged positions for a specific user address.

<ParamField path="address" type="string" required>
  The Ethereum address to get positions for
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/balancer/leverage/positions/0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/balancer/leverage/positions/0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/balancer/leverage/positions/0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
