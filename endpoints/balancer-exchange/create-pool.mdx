---
title: 'Create Pool'
api: 'POST /api/balancer/pools/create'
---

Create a new liquidity pool for a tokenized stock. If a pool doesn't exist for the specified stock, it will be created automatically.

<ParamField body="stockISIN" type="string" required>
  The ISIN of the stock to create a pool for
</ParamField>

<ParamField body="tokenAddress" type="string" required>
  The contract address of the tokenized stock
</ParamField>

<ParamField body="pairedToken" type="string">
  The address of the token to pair with (defaults to USDC)
</ParamField>

<ParamField body="weights" type="array">
  Array of weights for the pool tokens (defaults to [80, 20])
</ParamField>

<ParamField body="swapFee" type="string">
  The swap fee for the pool (defaults to "0.003")
</ParamField>

<RequestExample>
```bash cURL

  -H "Content-Type: application/json" \

  -d '{
    "stockISIN": "US0378331005",
    "tokenAddress": "0x1234567890123456789012345678901234567890",
```
