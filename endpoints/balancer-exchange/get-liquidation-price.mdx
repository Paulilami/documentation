---
title: 'Get Liquidation Price'
api: 'GET /api/balancer-exchange/leverage/liquidation-price/{positionId}'
---

Get the liquidation price and health metrics for a specific leveraged position.

<ParamField path="positionId" type="string" required>
  The unique identifier of the position
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/balancer-exchange/leverage/liquidation-price/position123" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/balancer-exchange/leverage/liquidation-price/position123', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/balancer-exchange/leverage/liquidation-price/position123',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
</RequestExample>

<ResponseExample>

{
