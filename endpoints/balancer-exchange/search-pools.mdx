---
title: 'Search Pools'
api: 'GET /api/balancer/v1/pools/search'
---

Search for pools by stock ISIN or token address.

<ParamField query="stockISIN" type="string">
  International Securities Identification Number
</ParamField>

<ParamField query="stockToken" type="string">
  Contract address of the stock token
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/balancer/v1/pools/search?stockISIN=US0378331005" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/balancer/v1/pools/search?stockISIN=US0378331005', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/balancer/v1/pools/search',
    params={'stockISIN': 'US0378331005'},
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)
```
