---
title: 'Search Pools'
api: 'GET /api/balancer/pools/search/{stockISIN}'
---

Search for a liquidity pool by stock ISIN. If a pool doesn't exist, it will be created automatically.

<ParamField path="stockISIN" type="string" required>
  The ISIN of the stock to search pools for
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/balancer/pools/search/US0378331005" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/balancer/pools/search/US0378331005', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/balancer/pools/search/US0378331005',
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)

data = response.json()
```
</RequestExample>
