---
title: 'Modify Leverage Position'
api: 'PATCH /api/balancer/v1/leverage/position/{positionId}/modify'
---

Modify an existing leveraged position by adding collateral or updating stop loss/take profit levels.

<ParamField path="positionId" type="string" required>
  The unique identifier of the position to modify
</ParamField>

<ParamField body="additionalCollateral" type="string">
  Additional collateral to add to the position
</ParamField>

<ParamField body="newStopLoss" type="string">
  New stop loss price (0 to remove)
</ParamField>

<ParamField body="newTakeProfit" type="string">
  New take profit price (0 to remove)
</ParamField>

<RequestExample>
```bash cURL
curl -X PATCH "https://trusser-issuer-backend.vercel.app/api/balancer/v1/leverage/position/12345/modify" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "additionalCollateral": "500",
    "newStopLoss": "185",
    "newTakeProfit": "225"
  }'
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/balancer/v1/leverage/position/12345/modify', {
```
