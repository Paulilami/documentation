---
title: 'Remove Liquidity'
api: 'POST /api/balancer/v1/pools/{poolId}/liquidity/remove'
---

Remove liquidity from a pool by burning BPT tokens.

<ParamField path="poolId" type="string" required>
  The unique identifier of the pool
</ParamField>

<ParamField body="maxBptAmountIn" type="string" required>
  Maximum amount of BPT tokens to burn
</ParamField>

<ParamField body="minAmountsOut" type="array">
  Minimum amounts of each token to receive [stockTokenAmount, pairedTokenAmount]
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/balancer/v1/pools/0x8b1e5c3d4a2f6b9e7c5d3a2f1b8e9c7d5a3f2b1e8c9d7a5f3b2e1c8d9a7f5b3e2c/liquidity/remove" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "maxBptAmountIn": "100",
    "minAmountsOut": ["20", "400"]
  }'
```
