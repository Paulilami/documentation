---
title: 'Initialize Pool'
api: 'POST /api/balancer/v1/pools/{poolId}/initialize'
---

Initialize a newly created pool with initial liquidity. This must be done before trading can begin.

<ParamField path="poolId" type="string" required>
  The unique identifier of the pool to initialize
</ParamField>

<ParamField body="exactAmountsIn" type="array" required>
  Array of exact token amounts [stockTokenAmount, pairedTokenAmount]
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/balancer/v1/pools/0x8b1e5c3d4a2f6b9e7c5d3a2f1b8e9c7d5a3f2b1e8c9d7a5f3b2e1c8d9a7f5b3e2c/initialize" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "exactAmountsIn": ["1000", "20000"]
  }'
```
