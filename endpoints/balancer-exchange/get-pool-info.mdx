---
title: 'Get Pool Information'
api: 'GET /api/balancer/v1/pools/{poolId}/info'
---

Get detailed information about a specific pool including metrics and current state.

<ParamField path="poolId" type="string" required>
  The unique identifier of the pool
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/balancer/v1/pools/0x8b1e5c3d4a2f6b9e7c5d3a2f1b8e9c7d5a3f2b1e8c9d7a5f3b2e1c8d9a7f5b3e2c/info" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/balancer/v1/pools/0x8b1e5c3d4a2f6b9e7c5d3a2f1b8e9c7d5a3f2b1e8c9d7a5f3b2e1c8d9a7f5b3e2c/info', {

```
