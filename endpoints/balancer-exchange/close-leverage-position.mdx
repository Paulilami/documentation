---
title: 'Close Leverage Position'
api: 'POST /api/balancer/v1/leverage/position/{positionId}/close'
---

Close an open leveraged position and realize profit/loss.

<ParamField path="positionId" type="string" required>
  The unique identifier of the position to close
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/balancer/v1/leverage/position/12345/close" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/balancer/v1/leverage/position/12345/close', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY'
  }
});
```

```python Python
import requests
```
