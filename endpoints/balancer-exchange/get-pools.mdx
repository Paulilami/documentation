---
title: 'Get All Pools'
api: 'GET /api/balancer/v1/pools'
---

Get a list of all pools created by the authenticated partner.

<ParamField query="active" type="boolean" default="true">
  Filter by active status
</ParamField>

<ParamField query="limit" type="number" default="100">
  Maximum number of pools to return
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of pools to skip for pagination
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://trusser-issuer-backend.vercel.app/api/balancer/v1/pools?active=true&limit=10" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://trusser-issuer-backend.vercel.app/api/balancer/v1/pools?active=true&limit=10', {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});
```

```python Python
import requests

response = requests.get(
    'https://trusser-issuer-backend.vercel.app/api/balancer/v1/pools',
    params={'active': True, 'limit': 10},
    headers={
        'x-api-key': 'YOUR_API_KEY'
    }
)
```
</RequestExample>
