---
title: 'Execute Swap'
api: 'POST /api/balancer/swap/execute'
---

Record the execution of a swap transaction. This endpoint is called after the user has successfully executed the swap transaction on-chain.

<ParamField body="txHash" type="string" required>
  The transaction hash of the executed swap
</ParamField>

<ParamField body="userAddress" type="string" required>
  The Ethereum address that executed the swap
</ParamField>

<ParamField body="expectedOutput" type="string">
  The expected output amount from the swap
</ParamField>

<ParamField body="stockISIN" type="string" required>
  The ISIN of the stock that was swapped
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://trusser-issuer-backend.vercel.app/api/balancer/swap/execute" \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "txHash": "0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef12",
    "userAddress": "0x742d35Cc6634C0532925a3b844Bc9e7595f5b5d4",
    "expectedOutput": "95.0",
    "stockISIN": "US0378331005"
  }'
```
