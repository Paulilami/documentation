---
title: 'Get Customer Trades'
description: 'Retrieve all executed trades for a specific customer'
api: 'GET /orderbooks/stocks/api/offchain/customers/{customerRef}/trades'
---

Returns a paginated list of trades where the specified customer was either the maker or taker, across off-chain order books.

## Authentication

Requires an instance API key with the `trading` service enabled.

## Path Parameters

<ParamField path="customerRef" type="string" required>
  The customer reference identifier. Max 256 characters.
</ParamField>

## Query Parameters

<ParamField query="limit" type="integer">
  Number of results to return. Default `50`, maximum `100`.
</ParamField>

<ParamField query="offset" type="integer">
  Offset for pagination. Default `0`.
</ParamField>

<RequestExample>
```bash
curl -X GET "https://api.trusset.org/orderbooks/stocks/api/offchain/customers/cust_12345/trades?limit=20" \
  -H "x-api-key: trusset_{instanceRef}_{secret}"
```
</RequestExample>
