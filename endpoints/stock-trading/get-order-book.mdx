---
title: 'Get Order Book'
description: 'Retrieve details of a specific order book'
api: 'GET /orderbooks/stocks/api/order-books/{orderBookId}'
---

Returns the full configuration and current status of a single on-chain order book.

## Authentication

Requires an instance API key with the `trading` service enabled.

## Path Parameters

<ParamField path="orderBookId" type="string" required>
  The order book identifier.
</ParamField>

<RequestExample>
```bash
curl -X GET "https://api.trusset.org/orderbooks/stocks/api/order-books/ob_abc123" \
  -H "x-api-key: trusset_{instanceRef}_{secret}"
```
</RequestExample>

| Code | Status | Description |
