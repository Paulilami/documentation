---
title: 'Get Issuer'
api: 'GET /api/mica/v1/issuers/{id}'
---

Get full details for a single MiCA entity by its ID.

## Path Parameters

<ParamField path="id" type="string" required>
  The entity ID
</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/api/mica/v1/issuers/clx1abc2d3efg" \
  -H "X-API-Key: trusset_mica_your_key_here"
```
```javascript JavaScript
const response = await fetch(
  'https://api.trusset.org/api/mica/v1/issuers/clx1abc2d3efg',
  { headers: { 'X-API-Key': 'trusset_mica_your_key_here' } }
);

const data = await response.json();
console.log(data.data.entityName);
```
```python Python
import requests

response = requests.get(
    'https://api.trusset.org/api/mica/v1/issuers/clx1abc2d3efg',
    headers={'X-API-Key': 'trusset_mica_your_key_here'}
)

data = response.json()
print(data['data']['entityName'])
```
</RequestExample>
