---
title: 'Health'
api: 'GET /api/mica/v1/health'
---

Check API operational status and data freshness.

<RequestExample>
```bash cURL
curl "https://api.trusset.org/api/mica/v1/health" \
  -H "X-API-Key: trusset_mica_your_key_here"
```
```javascript JavaScript
const response = await fetch(
  'https://api.trusset.org/api/mica/v1/health',
  { headers: { 'X-API-Key': 'trusset_mica_your_key_here' } }
);

const { data } = await response.json();
console.log(data.data.status);
```
```python Python
import requests

response = requests.get(
    'https://api.trusset.org/api/mica/v1/health',
    headers={'X-API-Key': 'trusset_mica_your_key_here'}
)

print(response.json()['data']['status'])
```
</RequestExample>
