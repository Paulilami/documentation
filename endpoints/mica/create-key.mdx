---
title: 'Create Key'
api: 'POST /api/mica/developer/keys'
---

Generate an additional API key. Maximum 3 active keys per account.

Requires your management token in the `Authorization` header.

<ParamField body="name" type="string">
  Optional label for the key (max 50 characters). Defaults to `"key"`.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.trusset.org/api/mica/developer/keys" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_MANAGEMENT_TOKEN" \
  -d '{"name": "production"}'
```
```javascript JavaScript
const response = await fetch('https://api.trusset.org/api/mica/developer/keys', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_MANAGEMENT_TOKEN'
  },
  body: JSON.stringify({ name: 'production' })
});

const data = await response.json();
console.log(data.data.apiKey);
```
```python Python
import requests

response = requests.post(
    'https://api.trusset.org/api/mica/developer/keys',
    headers={'Authorization': 'Bearer YOUR_MANAGEMENT_TOKEN'},
    json={'name': 'production'}
)

data = response.json()
print(data['data']['apiKey'])
```
</RequestExample>
