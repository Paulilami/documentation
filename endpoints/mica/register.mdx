---
title: 'Register'
api: 'POST /api/mica/developer/register'
---

Create a developer account and receive your first API key and management token.

<Warning>
  Save both the API key and management token immediately. They cannot be recovered after this response.
</Warning>

<ParamField body="name" type="string" required>
  Your name or company name (2â€“100 characters)
</ParamField>

<ParamField body="email" type="string" required>
  Your email address. One account per email.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.trusset.org/api/mica/developer/register" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acme Corp",
    "email": "dev@acme.com"
  }'
```
```javascript JavaScript
const response = await fetch('https://api.trusset.org/api/mica/developer/register', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    name: 'Acme Corp',
    email: 'dev@acme.com'
  })
});

const data = await response.json();
console.log(data.data.apiKey);
console.log(data.data.managementToken);
```
```python Python
import requests

response = requests.post(
    'https://api.trusset.org/api/mica/developer/register',
    json={
        'name': 'Acme Corp',
        'email': 'dev@acme.com'
    }
)

data = response.json()
print(data['data']['apiKey'])
print(data['data']['managementToken'])
```
