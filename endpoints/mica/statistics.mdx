---
title: 'Statistics'
api: 'GET /api/mica/v1/statistics'
---

Aggregate statistics including entity counts by license type, country, status, and the most recent authorizations.

<RequestExample>
```bash cURL
curl "https://api.trusset.org/api/mica/v1/statistics" \
  -H "X-API-Key: trusset_mica_your_key_here"
```
```javascript JavaScript
const response = await fetch(
  'https://api.trusset.org/api/mica/v1/statistics',
  { headers: { 'X-API-Key': 'trusset_mica_your_key_here' } }
);

const data = await response.json();
console.log(`Total entities: ${data.data.totalEntities}`);
```
```python Python
import requests

response = requests.get(
    'https://api.trusset.org/api/mica/v1/statistics',
    headers={'X-API-Key': 'trusset_mica_your_key_here'}
)

data = response.json()
print(f"Total entities: {data['data']['totalEntities']}")
```
</RequestExample>
