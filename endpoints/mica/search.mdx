---
title: 'Search'
api: 'GET /api/mica/v1/search'
---

Full-text search across entity names, LEIs, commercial names, and national authorities.

<ParamField query="q" type="string" required>
  Search query (min 2 characters, max 200)
</ParamField>

<ParamField query="limit" type="number" default="20">
  Maximum results to return (max 100)
</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/api/mica/v1/search?q=binance&limit=5" \
  -H "X-API-Key: trusset_mica_your_key_here"
```
```javascript JavaScript
const response = await fetch(
  'https://api.trusset.org/api/mica/v1/search?q=binance&limit=5',
  { headers: { 'X-API-Key': 'trusset_mica_your_key_here' } }
);

const data = await response.json();
data.data.forEach(issuer => console.log(issuer.entityName));
```
```python Python
import requests

response = requests.get(
    'https://api.trusset.org/api/mica/v1/search',
    headers={'X-API-Key': 'trusset_mica_your_key_here'},
    params={'q': 'binance', 'limit': 5}
)

data = response.json()
for issuer in data['data']:
    print(issuer['entityName'])
```
