---
title: 'List Keys'
api: 'GET /api/mica/developer/keys'
---

List all API keys associated with your developer account.

Requires your management token in the `Authorization` header.

<RequestExample>
```bash cURL
curl "https://api.trusset.org/api/mica/developer/keys" \
  -H "Authorization: Bearer YOUR_MANAGEMENT_TOKEN"
```
```javascript JavaScript
const response = await fetch('https://api.trusset.org/api/mica/developer/keys', {
  headers: { 'Authorization': 'Bearer YOUR_MANAGEMENT_TOKEN' }
});

const data = await response.json();
console.log(data.data.keys);
```
```python Python
import requests

response = requests.get(
    'https://api.trusset.org/api/mica/developer/keys',
    headers={'Authorization': 'Bearer YOUR_MANAGEMENT_TOKEN'}
)

data = response.json()
print(data['data']['keys'])
```
