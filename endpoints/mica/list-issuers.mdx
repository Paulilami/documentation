---
title: 'List Issuers'
api: 'GET /api/mica/v1/issuers'
---

Query the full MiCA register with filtering, sorting, pagination, and field selection.

<ParamField query="licenseType" type="string">
  Filter by license type. Comma-separated. Values: `CASP`, `ART`, `EMT`, `OTHER`, `NON_COMPLIANT`
</ParamField>

<ParamField query="country" type="string">
  Filter by country code. Comma-separated ISO codes (e.g., `DE,FR,AT`)
</ParamField>

<ParamField query="search" type="string">
  Full-text search across entity name, LEI, commercial name, and authority (max 200 chars)
</ParamField>

<ParamField query="status" type="string">
  Filter by status. Comma-separated. Values: `active`, `expired`, `non_compliant`
</ParamField>

<ParamField query="entityType" type="string">
  Filter by entity type. Values: `provider`, `issuer`, `whitepaper`, `non_compliant`
</ParamField>

<ParamField query="sortBy" type="string">
  Sort field. Values: `entityName`, `countryCode`, `licenseType`, `authorizationDate`, `status`
</ParamField>

<ParamField query="sortOrder" type="string" default="asc">
  Sort direction: `asc` or `desc`
</ParamField>

<ParamField query="limit" type="number" default="100">
  Results per page (max 500)
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of results to skip
</ParamField>

<ParamField query="fields" type="string">
  Comma-separated list of fields to return. Reduces payload size. Available: `id`, `entityName`, `entityIdentifier`, `licenseType`, `country`, `countryCode`, `nationalAuthority`, `authorizationDate`, `status`, `commercialName`, `address`, `website`, `websitePlatform`, `services`, `passportedCountries`, `comments`, `authorizationEndDate`, `entityType`
</ParamField>

<RequestExample>
```bash cURL
curl "https://api.trusset.org/api/mica/v1/issuers?licenseType=CASP&country=DE&limit=10" \
  -H "X-API-Key: trusset_mica_your_key_here"
```
```javascript JavaScript
const params = new URLSearchParams({
  licenseType: 'CASP',
  country: 'DE',
  limit: '10'
});

const response = await fetch(
  `https://api.trusset.org/api/mica/v1/issuers?${params}`,
  { headers: { 'X-API-Key': 'trusset_mica_your_key_here' } }
);

const data = await response.json();
console.log(`${data.pagination.total} total results`);
```
```python Python
import requests

response = requests.get(
    'https://api.trusset.org/api/mica/v1/issuers',
    headers={'X-API-Key': 'trusset_mica_your_key_here'},
    params={
        'licenseType': 'CASP',
        'country': 'DE',
        'limit': 10
    }
)

data = response.json()
print(f"{data['pagination']['total']} total results")
```
