---
title: 'Countries'
api: 'GET /api/mica/v1/countries'
---

Get all countries with MiCA-authorized entities, sorted by entity count.

<RequestExample>
```bash cURL
curl "https://api.trusset.org/api/mica/v1/countries" \
  -H "X-API-Key: trusset_mica_your_key_here"
```
```javascript JavaScript
const response = await fetch(
  'https://api.trusset.org/api/mica/v1/countries',
  { headers: { 'X-API-Key': 'trusset_mica_your_key_here' } }
);

const { data } = await response.json();
data.data.forEach(c => console.log(`${c.name}: ${c.count}`));
```
```python Python
import requests

response = requests.get(
    'https://api.trusset.org/api/mica/v1/countries',
    headers={'X-API-Key': 'trusset_mica_your_key_here'}
)

for c in response.json()['data']:
    print(f"{c['name']}: {c['count']}")
```
</RequestExample>
