---
title: 'License Types'
api: 'GET /api/mica/v1/license-types'
---

Get all MiCA license categories with entity counts.

<RequestExample>
```bash cURL
curl "https://api.trusset.org/api/mica/v1/license-types" \
  -H "X-API-Key: trusset_mica_your_key_here"
```
```javascript JavaScript
const response = await fetch(
  'https://api.trusset.org/api/mica/v1/license-types',
  { headers: { 'X-API-Key': 'trusset_mica_your_key_here' } }
);

const { data } = await response.json();
data.data.forEach(t => console.log(`${t.type}: ${t.count}`));
```
```python Python
import requests

response = requests.get(
    'https://api.trusset.org/api/mica/v1/license-types',
    headers={'X-API-Key': 'trusset_mica_your_key_here'}
)

for t in response.json()['data']:
    print(f"{t['type']}: {t['count']}")
```
</RequestExample>
